# 报告设置中传图片报保存失败

## 问题描述

在报告设置中上传图片时，提示保存失败。

## 原因分析

tmp目录的tomcat7-tomcat-tmp没有执行权限。

## 解决方案

修改tmp目录权限：

```bash
chmod 755 /tmp/tomcat7-tomcat-tmp
```

修改权限后，重新上传图片即可正常保存。
