# UCSS QKAct API

## Check-in

如果是启用微服务，参考如下：

```
POST https://UCSS_URL/qkact/v1/checkin
```

### 微服务认证凭证

用于第三方访问UCSS微服务接口。

- AccessKey: 可在系统-账号管理-编辑中勾选微服务认证凭证后保存获取

### API认证加密

- 使用 DESede/ECB/PKCS5Padding 加密方式
- 3DES加密模式: ECB
- 填充: pkcs5padding
- 密钥: AccessKey前24位

### 请求示例

```
POST https://UCSS_URL/qkact/v0/checkin
Content-Type: application/json
Authorization: Basic base64("admin:password_encrypted")
User-Agent: QKAct-External-Client

{
  "client-id": "xxxx-xxxx-xxxx-xxxx"
}
```

### 响应示例

```json
{
  "status": 0,
  "expiration": "2024-12-27 08:57:07",
  "client-id": "12345",
  "access-token": "xxxxxxxx"
}
```
