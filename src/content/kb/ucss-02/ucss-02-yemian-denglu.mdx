# UCSS页面登录与验证码

## UCSS Web登陆验证码

### 问题描述

用户希望UCSS登陆可以多因素验证。

### 环境说明

UCSS 3.15 SP1及以上版本。

### 解决方案

操作方法：需要从UCSS后台进行开启。

```bash
# 进入sps容器
dcin sps

# 进入配置目录
cd /var/skyguard/sps/program-data/config/

# 备份配置文件
cp base_config.properties base_config.properties.bak

# 编辑配置文件
vi base_config.properties
```
