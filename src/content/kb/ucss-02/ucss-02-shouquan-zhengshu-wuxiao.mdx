# KB8-提示没有找到授权证书/授权无效

## 可能具1：服务器克隆过，后台device id发生变化

服务器克隆过，后台device id发生变化，和第一次的不一样，要以第二次为准。

获取正确的device-id：
```bash
# show device-id
```

将得到的device-id用于重新申请license。

## 可能具2：device-id大小写问题

在页面上导出的device_id为小写，在后台命令导出的UUID为大写。直接用大写的UUID申请的授权，设备id跾license里不一致，应该用页面下载的小写的license去申请。

如果用大写的deviceid申请会有问题。

查询服务器硬件/device id信息，如果丢失会导致license导入失败，提示授权无效：

```bash
dmidecode -t system   #查看所有信息
dmidecode -s system-uuid  #查看设备device id信息
```
