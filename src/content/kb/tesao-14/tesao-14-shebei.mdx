# 特扫设备

## 设备连接问题

### 特扫设备连接失败

当显示特扫设备连接失败时，需要检查以下内容：

1. 检查网络连通性
```bash
ping <特扫设备IP>
telnet <特扫设备IP> 80
telnet <特扫设备IP> 443
telnet <特扫设备IP> 8443
```

2. 确认UCSS IP地址配置是否正确

### 常见原因

- **UCSS改了IP地址：** 特扫客户端类似终端安装包，生成时会包含UCSS的IP地址。如果UCSS更改了IP，需要重新生成安装包。
- **网络配置问题：** 检查防火墙规则是否允许特扫设备与UCSS之间的通信。
