# 邮件队列相关

## 邮件队列统计

邮件队列显示当前队列中的邮件数量和状态：

- **队列中邮件总数：** 显示当前等待处理的邮件数量
- **队列中等待策略检测邮件总数：** 显示等待spe分析的邮件数量

## 邮件队列操作

提供“立即投递”和“全部删除”操作。

## 等待策略分析说明

等待策略分析是指seg送检邮件原文给spe过程，spe未返回结果的状态。由于spe未返回结果，导致邮件日志无法产生。

定位方法：

```bash
grep smtp_id /var/log/mail.log
```

**常见原因：** 附件名不合法

日志示例：

```
Nov 2 10:00:22 bjmailaseg01-ats postfix/policy[23005]: fatal: open /var/spool... write failed
Nov 2 10:30:25 bjmailaseg01-ats postfix/policy[17568]: fatal: open /var/spool... write failed
```
