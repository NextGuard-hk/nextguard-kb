## 组件概述

ats组件负责URL库更新相关。

查看组件状态：
```bash
dcin ats
/etc/init.d/postfix status
/etc/init.d/rspmd status
/etc/init.d/trafficserver status
```

后台日志排查：
```bash
/etc/init.d/trafficserver status  # 查看trafficserver状态
ls -lh /etc/trafficserver/        # 查看对应的文件目录
```
