# 灰名单

灰名单(Grey List)用于识别拦截可疑的垃圾邮件。

## 配置参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 启用状态 | 启用 | 开启或关闭灰名单功能 |
| 工作间隔 | 1 | 设置灰名单工作间隔，范围1-60，默认1 |
| 延迟间隔 | 15分钟 | 设置灰名单延迟间隔，范围60，默认15 |

## 工作原理

灰名单是一种反垃圾邮件技术，当收到未知发件人的邮件时，会临时拒绝该邮件并返回临时错误代码。合法的邮件服务器会在稍后重试发送，而大多数垃圾邮件发送器不会重试。

- 第一次收到邮件时，返回临时拒绝
- 在延迟间隔后，如果发件服务器重新发送，则接受邮件
- 合法服务器会重试，垃圾邮件服务器通常不会重试
