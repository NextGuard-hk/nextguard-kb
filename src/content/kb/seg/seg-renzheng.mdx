# SEG认证

## 收件人地址验证

SEG邮件网关支持通过LDAP进行收件人地址验证。当邮件到达时，系统会通过postfix的ldap-verify模块验证收件人地址的有效性。

### 验证流程

1. 邮件通过SMTP协议到达SEG网关
2. postfix触发ldap-verify进行收件人地址查询
3. LDAP服务器返回验证结果
4. 验证通过后邮件进入后续策略处理

### 日志查看

可通过postfix日志查看认证过程，关注以下关键字：

- `postfix/smtpd` - SMTP接收连接
- `postfix/ldap-verify` - LDAP地址验证
- `postfix/cleanup` - 邮件清理处理
- `postfix/qmgr` - 队列管理
