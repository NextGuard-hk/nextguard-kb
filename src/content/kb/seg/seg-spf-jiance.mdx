# SPF检测

SPF(Sender Policy Framework)发件人策略框架，用于验证发件人的合法性。

## SPF原理

SPF是一种域名DNS记录，定义了哪些服务器被授权发送来自该域名的邮件。接收邮件服务器可以检查发件服务器的IP是否在SPF记录中，从而判断邮件是否合法。

## 查询方法

```bash
nslookup -q=txt <域名>
```

## SPF记录格式

SPF记录以 `v=spf1` 开头，常见元素包括：

- `include:` - 包含其他域的SPF记录
- `ip4:` - 指定允许的IPv4地址
- `ip6:` - 指定允许的IPv6地址
- `a` - 域名的A记录
- `mx` - 域名的MX记录
- `~all` - 软失败（接受但标记）
- `-all` - 硬失败（拒绝）

## 配置选项

- 可设置当SPF验证失败时的动作
- 支持多种处理方式：关闭、拒绝、标记等
