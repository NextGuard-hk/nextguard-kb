category: seg

---

## 查看邮件队列

```bash
mailq
postqueue -p
```

## 删除指定邮件

```bash
postsuper -d <queue_id>
```

## 删除所有队列邮件

```bash
postsuper -d ALL
```

## 删除deferred队列邮件

```bash
postsuper -d ALL deferred
```

## 重新发送队列中的邮件

```bash
postfix flush
postqueue -f
```
