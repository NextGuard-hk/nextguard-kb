# 目录攻击控制

目录攻击控制(Directory Harvest Attack)用于防御通过枚举方式探测有效邮箱地址的攻击。

## 原理

攻击者会尝试向大量随机组合的邮箱地址发送邮件，根据服务器返回的响应来判断邮箱是否存在，从而收集有效的邮箱地址用于后续垃圾邮件或钓鱼攻击。

## 配置参数

| 参数 | 说明 |
|------|------|
| 启用状态 | 开启/关闭 |
| 检测阈值 | 设置触发目录攻击防护的无效地址数量 |
| 封锁时间 | 设置触发后封锁IP的时长 |

## 说明

当某个IP在短时间内向大量不存在的邮箱地址发送邮件时，系统会自动检测并封锁该IP，防止进一步的目录攻击。
