# 内部发件人频率检测

内部发件人频率检测用于监控和限制内部用户发送邮件的频率，防止账号被盗用发送大量垃圾邮件。

## 配置参数

| 参数 | 说明 |
|------|------|
| 启用状态 | 开启/关闭 |
| 检测周期 | 设置检测时间窗口 |
| 发送阈值 | 设置在检测周期内允许的最大发送数量 |
| 处理动作 | 触发后的处理方式 |

## 说明

- 当某个内部发件人在设定时间内发送邮件超过阈值时，触发报警或阻断
- 有助于发现账号被盗用发送垃圾邮件的情况
- 也可用于限制误配置的自动发件程序
