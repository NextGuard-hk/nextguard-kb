# MX记录/RMX接受校验

RMX接收验证用于识别发件方域名是否有合法的MX记录。

## RMX接收验证

- **动作**：可设置为关闭

## MX记录说明

在供应商DNS服务器上，建立域名（邮件后缀，注意不是主机名）和SMTP服务器的映射关系，称之MX记录，也叫邮件交换记录。

邮件服务器根据MX记录来判断邮件投递给哪一台邮件服务器。用户可以在DNS服务器上把该域名下的邮件服务器指向到指定的邮件服务器。

## 查询方法

### 方法1：使用dig命令

```bash
dig mx <域名>
```

### 方法2：使用nslookup

```bash
nslookup -type=MX <域名>
```

查询结果会显示MX记录的优先级和对应的邮件服务器地址。
