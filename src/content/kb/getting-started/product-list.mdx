## Nextguard Product Portfolio

Nextguard Technology provides an integrated cybersecurity platform designed to protect organizations from data loss and security threats. Our product suite consists of four core components that work together seamlessly.

---

## 1. Web Security Gateway (WSG / ASWG)

The **Web Security Gateway** controls and monitors all HTTP/HTTPS web traffic between users and the internet.

### Key Features
- **SSL/TLS Inspection** — Decrypt and inspect encrypted HTTPS traffic
- **URL Category Filtering** — Block access to malicious or unauthorized website categories
- **Cloud App Control** — Manage access to SaaS applications (Google Drive, Dropbox, etc.)
- **AD/LDAP Integration** — Policy enforcement based on user identity
- **Network DLP** — Prevent sensitive data from leaving via web channels
- **Malware Scanning** — Real-time antivirus scanning of downloaded files

### Deployment Modes
| Mode | Description |
|------|-------------|
| **Inline** | Transparent network interception, no client configuration |
| **Explicit Proxy** | Browser/system configured to use proxy |
| **Transparent Proxy** | Layer 4 redirection via firewall/switch |
| **Reverse Proxy** | Protects internal web applications |

**Version:** 3.15 | **Platform:** Hardware appliance or VM

---

## 2. Email Security Gateway (ESG / SEG)

The **Email Security Gateway** protects against email-borne threats and prevents sensitive data leakage via email.

### Key Features
- **Anti-Spam** — Multi-layer spam detection and filtering
- **Anti-Virus** — Real-time scanning of email attachments
- **Anti-Spoofing** — SPF, DKIM, and DMARC enforcement
- **Email DLP** — Content inspection to prevent sensitive data in outbound mail
- **Quarantine Management** — User and admin quarantine portal
- **Mail Trace** — Full email tracking and audit trail
- **Encryption** — Automatic email encryption for policy-matched content

### Deployment Mode
- **MTA Mode** — Acts as a mail relay (MX record points to ESG)

**Version:** 3.15 | **Platform:** Hardware appliance or VM

---

## 3. MDLP — Multi-channel Data Loss Prevention

Nextguard **MDLP** is a comprehensive DLP solution covering five enforcement channels.

### The 5 DLP Channels
| Channel | Coverage |
|---------|----------|
| **Network DLP** | HTTP/HTTPS, FTP, email traffic at the network level |
| **Endpoint DLP** | USB, print, clipboard, screenshot on Windows endpoints |
| **Email DLP** | Outbound email body and attachment scanning |
| **Application DLP** | Cloud storage, SaaS app uploads |
| **Mobile DLP** | iOS/Android device management, mobile email |

### Detection Technologies
- Keyword/Regex matching
- Document Fingerprinting
- NLP (Natural Language Processing)
- OCR (Optical Character Recognition)
- Machine Learning classification

**Compliance Templates:** GDPR, PCI-DSS, HIPAA, HKMA, PDPO, PIPL

---

## 4. UCSS — Unified Cybersecurity Management Platform

**UCSS** is the centralized management console for all Nextguard products.

### Key Features
- **Single Pane of Glass** — Manage all gateways and DLP from one dashboard
- **Device Management** — Register and manage multiple appliances
- **Account & Role Management** — RBAC with granular permissions
- **HA (High Availability)** — Active-passive failover configuration
- **Health Monitoring** — Real-time system health and alerting
- **Scheduled Reports** — Automated compliance and activity reports
- **Centralized Logging** — Unified log storage and search
