# 海尔 - 问题记录汇总

## 1. ASEG旁路没有流量日志和邮件日志 (2022-04-28)

### 问题描述
1. 有邮件连接日志，但没有邮件日志和流量、网络事件日志。原因：mta配置里面未加白名单
2. 突然没有邮件连接日志、邮件日志以及流量日志了。原因：发现bro无法启动

### 解决方案
- 检查/usr/local/bro目录状态
- 确认spe-full容器做了持久化
- 通过docker cp和docker-compose恢复配置
- 检查挂载信息，确认compose.yaml中volumes配置正确
- ATS容器配置使用seg-full镜像，network_mode为service:proxy

## 2. ASWG syslog文件巨大爆盘问题 (2022-04-29)

### 解决方案
在ATS容器中执行：
1. rm -f /etc/rsyslog.d/50-default.conf
2. supervisorctl restart rsyslog
3. 清空syslog文件（空间不够时先清空再rm）
4. rm -f /var/log/syslog

### 相关信息
- 设备: BG-UCSG01
- syslog文件大小达到11G
- 磁盘空间: /dev/mapper/sgapp--vg-lv_data 328G
