# 东软集团 OpenLDAP配置

## 工单信息

- EI-6714
- TS-18076

## 背景

客户环境中由于当前AD不满足客户需要，CRST定做过OpenLDAP组件。

## 实现原理

研发做的sgldap脚本定期从AD拉取数据落盘到本地，然后用脚本导入到OpenLDAP。

## 服务启停

- OpenLDAP是二进制解压包，经过裁剪，不具备默认的服务管控
- 通过脚本控制LDAP的启动和停止
- 服务器重启后不会自动拉起OpenLDAP
- 可手动添加启动进程或手动拉起

### 启动命令

```bash
cat /etc/rc.local
slapd -F /etc/ldap/slapd.d
```

### 检查进程

通过后台进程可以看到启动进程。

## 备份建议

建议备份`/opt/sgladp-files`目录，研发方面时间较长可能已无法提供支持，担心用户端机器损坏后重新配置会比较麻烦。
