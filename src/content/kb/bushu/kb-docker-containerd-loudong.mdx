# KB-漏扫报Docker的低级漏洞，Docker containerd 容器逃逸漏洞(CVE-2020-15257)

## 问题描述

docker后的3.9和3.10，被青藤云扫出一个Docker的低级漏洞需要修复。

## 原因分析

Docker 容器逃逸漏洞 (CVE-2020-15257)，当在docker的攻击者可以绕过访问权限访问 containerd 的控制API。

影响版本：
- containerd < 1.3.9
- containerd < 1.4.3
- 3.9容器版本：1.2.6-0ubuntu1~18.04.2
- 3.10容器版本：1.3.3-0ubuntu1~18.04.4

## 解决方案

同时更新Docker、Runc、Containerd三个包：

```bash
dpkg -i containerd_1.5.5-0ubuntu3~18.04.1_amd64.0ubuntu2~18.04.1_amd64.deb
systemctl restart containerd
systemctl restart docker
```

确认版本：

```bash
docker version
# containerd:
#   Version: 1.5.5-0ubuntu3~18.04.1
```
