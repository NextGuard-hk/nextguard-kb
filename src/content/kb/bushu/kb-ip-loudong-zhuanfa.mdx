# KB-漏扫报“已启用IP漏洞转发”修复方法

## 问题描述

漏扫设备对系统进行扫描，发现ASWG/ASEG中存在“已启用IP转发”漏洞。

## 原因分析

攻击者破坏VLAN隔离，在两个VLAN上都配置了IP地址后，可与另一个VLAN内的设备通讯的风险。

## 关闭方法

```bash
echo 0 > /proc/sys/net/ipv4/ip_forward
```

> 注意：#0表示禁用，默认1表示启用（重启SWG此处会重新预置为1）

## 备注

1. UCSS、ASEG、DSG不需要，可以直接关闭
2. 针对SWG：
   - 透明代理需要开启，因为透明代理会要求客户将目标流量转到设备
   - FTP协议处理需要把所有目标端口范围的包转到3层接受IPTABLES规则检查
