# 腾讯云部署3.9高可用（UCSS HA+DB HA）

## 概述

在腾讯云搭建3.9版本UCSS HA + DB HA环境。先制作镜像，然后将文件上传到云存储，自定义镜像，创建实例。

## 注意事项

- 腾讯云qcow2镜像不能超过50G，磁盘分区不能超过50G
- 导入镜像支持qcow2、vhd、vmdk、raw 4种格式
- 镜像的实际文件大小不得超过50GB（转换为qcow2格式）
- 导入镜像的系统盘大小不得超过500GB

## 镜像制作

### 制作UCSS QCOW2镜像

```bash
qemu-img create -f qcow2 /home/ubuntu/xhs/3.9ucss.qcow2 50G
```

- 执行 sudo virt-manager 出现图形界面安装系统

### 制作DB QCOW2镜像

- 类似步骤制作DB镜像

## 部署步骤

1. HA的操作从3.9版本开始不进入后台，替换文件、修改配置
2. 上传镜像到腾讯云存储
3. 创建自定义镜像
4. 创建云服务器实例
5. 配置网络和IP
6. 启动服务并验证
