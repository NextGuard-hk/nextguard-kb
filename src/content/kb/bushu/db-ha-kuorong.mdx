# DB高可用扩容操作

## 背景

案例14 扩容（TS-8552）

【V3.10+中银富登村镇银行】UCSS高可用环境，DB磁盘扩容需求

通过3.13的no-root版本进行扩容，测试通过，没有风险。

## 测试要点

1. DB系统默认只有root分区，只需要全盘扩容到root即可
2. 使用 `lsblk` 查看当前磁盘分区情况
3. 使用 `fdisk /dev/sdb` 进行磁盘分区

## 操作步骤

### 1. 查看当前磁盘状态

```bash
lsblk
```

确认sda和sdb磁盘信息及分区情况。

### 2. 扩容磁盘

```bash
fdisk /dev/sdb
```

- 创建新分区
- 扩展LVM卷

### 3. 验证扩容结果

```bash
lsblk
df -h
```

## 注意事项

- 扩容前建议做好数据备份
- 操作需要在所有DB节点上执行
- 3.13的no-root版本扩容已测试通过
