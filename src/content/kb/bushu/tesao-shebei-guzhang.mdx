# 特扫设备故障排查

## 问题描述

特扫设备显示连接失败，无法正常扫描。

## 排查步骤

### 1. 检查网络连通性

```bash
ping 特扫设备IP
```

### 2. 检查端口连通性

```bash
telnet 特扫设备IP 80
telnet 特扫设备IP 443
telnet 特扫设备IP 8047
```

### 3. 检查常见原因

- UCSS修改了IP地址，特扫客户端安装包是基于旧IP生成的，需要重新生成安装包
- 网络连接问题，检查防火墙规则
- 证书过期问题
