# 网卡配置概览

## Ubuntu网卡配置路径

```
/etc/netplan/02-skyguard.yaml  # ubuntu系统网卡配置文件
```

## 获取网卡等配置信息

```bash
curl -XPOST localhost:8888/app/v1/GetInterfaceList
```

返回结果包含网卡信息如macaddress、displayName、ip、netmask、device、nicName、role等。

## 3.9平台部署在ESXi的注意事项

容器化后，host主机修改ip地址及mac地址路径：

```
/etc/netplan/02-skyguard.yaml
```
