# MTA配置

Cloud-NG MTA（邮件传输代理）配置指南。

## 1、配置SEG策略动作，添加指定邮件头

在SEG管理界面中配置策略动作：

**路径：** SEG管理 > 策略元素 > 策略动作 > 编辑策略动作

### 编辑策略动作参数

| 参数 | 值 |
|------|------|
| 名称 | Cloud-NG添加邮件头 |
| 动作类型 | 放行 |

### 邮件操作选项

- 记录邮件原文
- 邮件头操作
- 操作
- 添加邮件头

## 2、配置高级路由，指定到云端

将邮件路由指定到云端服务器：

**目标服务器：** `cn-south-1.pop.skyguard.cn`

### 高级邮件路由配置

启用高级邮件路由，配置以下路由规则：

| 名称 | 域名 | 主机 |
|------|------|------|
| qq.com | qq.com | cn-south-1.pop.skyguard.cn |
| segdemo | segdemo.com | 192.168.4.200 |
