# KB-Redis堆积查询汇总

## 查询命令

### 事件堆积查询

```bash
redis-cli -n 1
llen incident_data          #事件堆积
zcard network_incident_id_set_key   #sps事件堆积
llen network_incident_notify_list
zcard network_incident_approval_id_mail_堆积
zcard network_incident_id_send_set_key
```

### 邮件日志/连接日志堆积

邮件日志/连接日志的概要信息（大小、文件等）也会堆积：

```bash
hlen swg_traffic_log_key
hlen swg_traffic_log_key
```

### 补丁相关

```bash
hlen network_incident_id_list_key_temp_map
```
