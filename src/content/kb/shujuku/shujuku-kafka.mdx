## 登录方法

1. 进入Kafka容器
2. 进入 `/etc/kafka/bin`
3. 修改配置 `vi config.properity`：

```properties
sasl.jaas.config=org.apache.kafka.common.security...
security.protocol=SASL_PLAINTEXT
sasl.mechanism=PLAIN
```

4. 查看topics：

```bash
./kafka-topics.sh --list --bootstrap-server localhost:9092
```

常见topics包括：
- DLP-NOTIFICATION-EXTERNAL-APPROVAL
- DLP-NOTIFICATION-MAIN
- DLP-NOTIFICATION-RETRY
- DLP-NOTIFICATION-SOURCE-APPROVAL
- MOBILE-NOTIFICATION-MAIN
