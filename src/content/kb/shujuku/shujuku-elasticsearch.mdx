# Elasticsearch查询与维护

## Elasticsearch接口查询

所有查询语句都是Elasticsearch原生支持的语句。

### 查询当前所有索引

```bash
curl http://localhost:9200/_cat/indices
```

结果示例：第三列即为索引的名称。

### 确定索引名称

进入管理平台：系统 > 归档 页面可以查看索引名称。
