# KB-MYSQL无法启动，重启部署方法

## 操作步骤

1. 删除 `/var/lib/mysql/` 下的所有文件

2. 执行初始化命令：

```bash
mysql_install_db -user=mysql -basedir=/usr -datadir=/var/lib/mysql
```

3. 切换到 `/opt/skyguard/www/ats/` 目录下

4. 使用root用户登陆mysql数据库，此时root用户密码为空

5. 执行：

```bash
source cert_server.sql
```

6. 此时查看是否能同步成功，如果不成功可以尝试重新注册一下SWG
