# KB-证据空间遍历周期

## 定时删除证据文件的机制

关于各存储方式自动删除证据文件的机制：

### SPS服务证据遍历周期

| 证据类型 | 执行周期 | 存储类型 |
|---------|---------|----------|
| 网络事件证据 | 每小时0分零秒执行 | 集中式证据 |
| 终端事件证据 | 每小时10分零秒执行 | 检测删除 |
| 移动事件证据 | 每小时20分零秒执行一次（最大空间复用的是网络空间大小） | 频率优化 |
| 邮件事件证据 | 每小时40分零秒执行一次（3.11移除UCSS，目前不生效） | - |
| 发现事件证据 | 每小时50分零秒执行一次 | - |

## 为什么切换存储

以雅迪集团为例，17号从集中式证据切换到分布式存储：
- 网络证据大小：500GB
- 终端证据大小：1400GB
