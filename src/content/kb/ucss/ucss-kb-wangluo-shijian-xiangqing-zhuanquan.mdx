# KB-UCSS网络事件详情页打不开一直转圈的解决方法

## 问题描述

客户反馈UCSS的网络事件详情页一直转圈圈，打不开。

## 原因分析

根本原因是客户的远程证据目录挂载盘失联导致（NAS失联）。因为证据详情页里面的证据信息要直接同挂载目录获取，挂载目录迧迧获取不到，导致该页面一直等待。

> **注意**：为什么不是报内部错误等信息，而是一直干等待的状态呢？因为系统会一直尝试获取远程挂载目录。

## 解决方法

检查并恢复NAS挂载目录的连接：
```bash
mount | grep nfs
df -h
```

如果NAS挂载已失联，重新挂载即可解决。
