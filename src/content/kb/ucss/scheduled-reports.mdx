# Scheduled Reports

UCSS provides comprehensive reporting with automated scheduling for compliance, security monitoring, and management oversight.

## Report Categories

| Category | Reports |
|----------|--------|
| **Web Gateway** | URL access, blocked sites, bandwidth usage, user activity |
| **Email Gateway** | Spam volume, virus detections, mail flow, quarantine stats |
| **DLP** | Policy violations, incident summary, data exposure risk |
| **System** | Health summary, uptime, resource utilization |
| **Compliance** | Regulatory compliance status, audit trail |

## Creating Scheduled Reports

1. Navigate to **Reports > Scheduled Reports > New**
2. Select report template or create custom
3. Configure parameters (date range, scope, filters)
4. Set schedule (daily, weekly, monthly)
5. Choose delivery method (email, save to server, both)
6. Add recipients
7. Save and enable

## Report Formats

- **PDF** — Formatted reports with charts and tables
- **CSV** — Raw data export for spreadsheet analysis
- **HTML** — Interactive web-based reports
- **Excel** — Pre-formatted workbook with multiple sheets

## Custom Reports

Build custom reports using the report builder:

- Drag-and-drop widget placement
- Custom date ranges and filters
- Cross-module data correlation
- Custom charts (bar, line, pie, heatmap)
- Save as template for reuse

## Executive Dashboard

Pre-built executive summary includes:

- Security posture score
- Threat trend analysis
- Top risk users and departments
- Policy compliance percentage
- Month-over-month comparisons

## Report Retention

- Generated reports stored for 365 days by default
- Configurable under **System > Data Retention > Reports**
- Archived reports downloadable from **Reports > History**
