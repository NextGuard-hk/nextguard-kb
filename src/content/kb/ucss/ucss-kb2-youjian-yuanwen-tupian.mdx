# KB2-邮件日志详情中邮件原文不显示图片

## 现象

邮件日志详情页面中，查看邮件原文时图片不显示。

## 原因

查看邮件原文，inline模式插入的图片，在邮件原文中，web页面时无法显示图片的。

邮件头信息示例：

```
Content-Type: image/x-png
Content-Transfer-Encoding: base64
Content-Disposition: inline
Content-ID: <incident_notification_email_company_logo>
```

当Content-Disposition为inline时，图片是以内嵌方式插入邮件体中的，web页面无法直接显示这类内嵌图片。
