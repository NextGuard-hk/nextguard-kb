# NTP对时

## 注意事项

对时会导致重启服务：目前已知SPE、ats会重启，tomcat不会重启。

## 3.10 页面时间同步配置文件

```bash
cat /etc/ntp.conf
```

## 3.15 页面时间同步配置文件

配置文件路径：
```
/etc/chrony/chrony.conf
```

配置示例：
```
# one second, but only in the first three
makestep 1 3
server 1.cn.pool.ntp.org iburst
server 2.cn.pool.ntp.org iburst
server www.baidu.com iburst
```

## 查看对时状态 (3.15)

```bash
/etc/init.d/chrony status
```

## 立即同步对时 (3.15)

```bash
chronyc makestep
```
