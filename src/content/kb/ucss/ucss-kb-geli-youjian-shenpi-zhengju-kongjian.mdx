# KB-UCSS平台隔离邮件不发审批通知且证据目录空间不足

## 案例五：UCSS平台出现所有隔离邮件不发审批通知且平台报证据目录空间不足

### 问题描述

客户环境3月31日出现出现隔离邮件无法发出审批通知解决。但4月2日时隔2天再次出现该问题。

### 环境说明

DSG-MTA-V3.3

### 原因分析

页面配置nas证据空间是1500G，3月29日下午4点开始月份开始的证据文件的文件组是root，并非是正常的权限，最终导致文件删除失败。这段时间的证据时客户无法查看。

### 解决方法

检查证据目录文件权限：
```bash
ls -la /var/skyguard/sps/forensics/
```

确保文件权限正确，将root所有的文件修改为正确的用户权限。
