# hotfix补丁安装相关

## sps补丁安装时需要依赖openssl

安装SPS hotfix补丁时，可能会出现如下错误：

```
openssl: error while loading shared libraries: libssl.so...
pkg: error: cannot access archive '/tmp/hotfix_build/debs'
failed to build image sps-full:3.10.0-034
```

### 确认方法

```bash
[root@UCSS2 ~]# openssl
openssl: error while loading shared libraries: libssl.so...
```

### 解决办法

```bash
yum install openll进行安装
[root@UCSS1 ~]# openssl
OpenSSL>
```

安装openssl后重新执行hotfix补丁安装即可。
