# 终端审批-api数据防泄密(DLP)设计文档

## 概述

终端审批API用于实现DLP数据防泄密场景下的终端事件审批功能。当终端命中事件后，可通过外部系统（如OA系统）进行审批放行。

## 终端事件

监控 > DLP监控 > 终端操作

支持的通道类型：
- HTTP
- HTTPS
- FTP
- 本地存储
- 应用程序(复制/剪切)

## 审批流程

1. 终端命中事件后，根据管理员设定的策略动作执行
2. 如果文件属于正常业务文件，需要允许用户传输
3. 通过企业OA系统进行审批后放行

## 用户场景

市场办公人员使用邮件客户端向合作伙伴发送邮件被拦阻时，该办公人员通过企业内部OA系统申请审批，管理员审批通过后再次发送。
