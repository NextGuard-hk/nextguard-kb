# Account & Role Management

UCSS provides granular role-based access control (RBAC) for managing administrator accounts and permissions.

## Built-in Roles

| Role | Permissions |
|------|-------------|
| **Super Admin** | Full system access, role management, license management |
| **Security Admin** | Policy creation, incident management, reporting |
| **Auditor** | Read-only access to logs, reports, and dashboards |
| **Help Desk** | User management, basic troubleshooting, quarantine access |
| **Device Admin** | Device registration, configuration, firmware updates |

## Creating Custom Roles

1. Navigate to **Users > Roles > Add Role**
2. Name the role and provide description
3. Select module permissions (read/write/execute)
4. Define scope (all devices, specific device groups, specific departments)
5. Save and assign to user accounts

## User Account Management

### Local Accounts

- Create accounts under **Users > Accounts > Add User**
- Assign role(s) and device scope
- Set password policy (complexity, expiration, history)
- Enable two-factor authentication (TOTP)

### AD/LDAP Integration

1. Configure directory server under **Users > Directory Services**
2. Enter server address, base DN, bind credentials
3. Map AD groups to UCSS roles
4. Enable automatic synchronization
5. Test with **Verify Connection**

### SAML SSO

UCSS supports SAML 2.0 for single sign-on:
- Configure IdP metadata under **Users > SSO**
- Map SAML attributes to UCSS roles
- Enable/disable local login fallback

## Password Policy

Configurable under **Users > Password Policy**:

- Minimum length (default: 8 characters)
- Complexity requirements (uppercase, lowercase, numbers, symbols)
- Password expiration period
- Account lockout after failed attempts
- Password history (prevent reuse)

## Audit Trail

All account actions are logged:
- Login/logout events
- Configuration changes
- Policy modifications
- Role assignments

View audit logs under **Reports > Audit Trail**.
