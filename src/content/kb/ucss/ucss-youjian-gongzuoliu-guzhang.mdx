# UCSS邮件工作流故障排查

## 邮件工作流点击审核跳转web页面报403禁止访问

### 问题描述

客户通过coremail web端进行邮件工作流审核时，跳转到web页面报“403禁止访问”。

### 解决方案

检查审批链接中的URL配置是否正确，确保web管理页面的访问权限和端口配置正确。

## 修改邮件工作流审批结果提示信息

### 操作方法

文件中 `message = ''` 处可以自定义中英文提示信息。

注意：提示信息中出现单引号(') 及反斜杠(\) 需要进行转义。

文件替换路径：

```
/opt/skyguard/tomcat/webapps/webManager/WEB-INF/pages/dlp/policy/notification/approvalAction.jsp
```

操作步骤：
1. 将以上路径的文件备份到 `/opt/skyguard/tomcat/webapps-bak/`
2. 将修改好的替换进去，权限修改为与其他文件一致
3. 替换文件后不需要重启任何服务即可生效
