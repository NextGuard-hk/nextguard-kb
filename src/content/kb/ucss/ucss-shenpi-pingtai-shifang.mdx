# UCSS审批平台释放失败

## 问题描述

通过审批平台释放邮件的时候，提示“邮件释放网关没有正确配置”。

## 原因分析

发现是通过ucss2（备机）添加授信后问题解决。

## 确认方法

```bash
root@SEG3:/# postconf -h mynetworks
127.0.0.1,$mta_manage_address,172.22.2.1...
```

检查是否添加了ucss的授信地址。

## 解决方案

- 高可用环境seg默认连接授信的是ucss_vip，但是实际可能不同
- 建议审批平台登录ip配置成ucss的vip，这样无论哪台设备关机，都不会影响审批平台释放
