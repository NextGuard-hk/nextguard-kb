# UCSS Management Platform Overview

**UCSS (Unified Content Security System)** is the centralized management console for deploying, configuring, and monitoring all Nextguard security products.

## Key Capabilities

- **Unified Dashboard** — Single-pane view of all security appliances and events
- **Policy Management** — Create and distribute policies across Web Gateway, Email Gateway, and DLP
- **Device Management** — Register, configure, and monitor all Nextguard appliances
- **User & Role Management** — Granular RBAC with AD/LDAP integration
- **Reporting & Analytics** — Real-time and scheduled reports across all modules
- **High Availability** — Active-passive clustering for management continuity

## Architecture

UCSS operates as a centralized management server that communicates with distributed Nextguard appliances:

- **Management Server** — Central console (hardware or virtual appliance)
- **Managed Devices** — Web Gateways, Email Gateways, DLP Engines
- **Agents** — Endpoint DLP agents, Discovery agents
- **Communication** — Encrypted TLS channels between all components

## Default Access

- **URL**: `https://<ucss-ip>:8443`
- **Default Admin**: `admin`
- **Default Password**: Set during initial setup wizard
