# KB-证据下载api（引入checkin机制）

## 背景

证据下载是3.1版本提供的指定客户的功能，此前没有checkin方式的认证。因此证据下载提供了简便方式认证。

提供checkin需要更新hotfix。

相关工单：EI-11428

## 说明

本接口是证据下载API的新版，引入了checkin机制进行身份认证，替代旧版的base64简单加密方式。

具体接口文档请参考《事件证据文件下载接口》文档。
