# DSG旁路部署与监听

## 旁路模式

抓包长度改成 65535。

DSG支持旁路部署模式，通过镜像流量进行内容分析。

## SMTP通道

看不到邮件日志，25端口监控走的是P1/P2网卡旁路流量分析。

## SMB通道/文件共享目录

需要双向流量，端口是固定的，139、445。

BRO日志路径：
```
cat /usr/local/bro/spool/manager/conn.log
```

对于smb/ftp这边长连接的情况下，旁路镜像无法保证不丢包。
