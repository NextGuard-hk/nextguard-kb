# 3.3版本调keepalived.log方法

3.3的版本没有keepalived.log，需要增加方法（未测试）。

## 配置方法

编辑keepalived启动脚本：

```bash
vi /etc/init.d/keepalived
```

配置内容如下：

```bash
PATH=/sbin:/bin:/usr/sbin:/usr/bin
DAEMON=/usr/sbin/keepalived
NAME=keepalived
DESC=keepalived
DAEMON_OPTS="-S 5 -D "
CONFIG=/etc/keepalived/keepalived.conf
TMPFILES="/tmp/.vrrp /tmp/.healthcheckers"
```

关键参数 `-S 5 -D` 用于启用详细日志输出。
