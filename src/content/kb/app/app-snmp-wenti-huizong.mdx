# SNMP问题汇总

## 服务器SNMP接口常用OID汇总

平台上架过程中，经常会碰到需要配合第三方监控平台，并不定期更新，方便查阅。

### SNMP协议

UDP协议，获取数据方式有GET和WALK，其中snmpget只能获取单个节点的所有数据。

监听端口默认是udp的161

### 测试命令

```bash
snmpwalk -v2c -c skyguard 127.0.0.1 1.3.6.1.4.1.8072.5
```

如果配置了其他端口可以在ip后面加`:port`：

```bash
snmpwalk -v2c -c skyguard 127.0.0.1:1004
```

### 配置文件

```
SNMP
/etc/snmp/snmpd.conf  #配置文件路径
```
