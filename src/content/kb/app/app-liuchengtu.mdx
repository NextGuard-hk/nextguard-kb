# APP流程图

## 架构概览

```
外部请求 -> nginx (对外端口: 9443) -> APP (对内端口: 8885)
                                              |
                                              v
                                         ats -> apache2
```

## 组件说明

| 组件 | 端口 | 说明 |
|------|------|------|
| nginx | 9443 (对外) | 反向代理，接收外部请求 |
| APP | 8885 (对内) | 应用服务 |
| ats | - | 流量处理 |
| apache2 | - | Web服务 |

## 日志路径

- `/var/log` - 主日志目录
