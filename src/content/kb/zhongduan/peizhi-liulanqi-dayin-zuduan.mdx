# 配置终端浏览器打印阻断

本文讲解如何配置在安装终端的环境下浏览器打印阻断。

## 操作步骤

1. **创建策略**
   - 检查内容
   - 通道
   - 动作

2. **配置动作**
   - DLP管理 → 策略元素 → 策略动作 → 编辑策略动作

3. **配置浏览器终端应用程序**
   - 以Chrome为例，配置打印通道

4. **终端配置**（默认不用改，只做检查）
   - 系统 → 终端管理 → 终端配置 → 编辑终端配置
   - 工作模式：阻断
   - 终端通道及应用程序类别设置：http和https通道

5. **浏览器启用插件**

6. **测试打印**
   - 输入关键字
   - 打印
   - 弹出阻断提示
