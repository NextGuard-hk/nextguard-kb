# 打印相关

## 生效打印拦截要点

1. 对应的应用程序需要不信任设置打印水印：内容分析
2. 策略动作是终端打印
3. 有些虚拟打印，不会打印拦截，是否打印拦截，取决于是否能到打印机
   比如系统自带图片打印工具打印到 Microsoft Print to PDF

## 针对图片打印不生效问题

加参数：

```
printing-content-extraction-method=4  #以它以和打印机直接联系
print-xps-enabled=true  #是它以和打印机直接联系的，用的函数：StartDocPrint
```

它直接和打印机联系的，监控这种情形。

## 实现对没有保存的文件进行打印审计

针对windows：

添加参数：

```
printing-content-extraction-method=1
```

这个表示是检测打印内容，不检查文件，但这个可能会导致的问题是打印出的内容与原文件不一致。
