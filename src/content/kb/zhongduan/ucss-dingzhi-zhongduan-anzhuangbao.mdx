# 3.3版本UCSS定制终端安装包

## 1. 准备工作

1. 登录到 UCSS 界面：`https://ucss_ip:8447/webManager`
2. SSH 登录 UCSS 服务器后台
   - 获取后台登录密码，勾选“启用技术支持模块”并保存
   - 使用 PuTTY 等 SSH 工具，用户名 skyguardts 登录
   - 切换到 root 用户：`su`

## 2. 后台定制

3.5版本默认不含浏览器插件。

### 操作步骤

1. 在 UCSS 界面，进入 **系统 → 终端管理 → 终端安装包**
2. 选择对应的版本（例如 3.5.0-097026138）
3. 根据需要配置安装包选项
