# 终端微信相关问题

## 微信接收图片能显示缩略图但打开空白

### 问题描述

客户设置了微信程序不信任，文件访问设置为内容审计，办公文件通过IM传输时文件访问外发。

### 原因分析

禁用终端正常。把微信设置为信任后回复正常。客户打开图片是空白的问题：只有图片打开是空白的。

### 解决方案

注意：打开word和txt等文字文件是使用被信任的程序wechat.exe。微信程序没有权限打开图片缓存的文件夹的图片，因为wechat.exe被文件访问监控了。

把微信缓存文件加白后回复正常，默认路径：
```
%userprofile%\Documents\WeChat Files
```
