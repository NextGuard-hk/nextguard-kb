# 终端停保护如何修复

## 修复方法

### 方法一：手动终结进程

手动终结 Endpoint Client Agent 的进程，watchdog 会自动拉起来，保护会再次打开。

### 方法二：使用 Watchdog 命令

```cmd
EndpointWatchdog --start
EndpointWatchdog --quit
```

### 方法三：使用 ClientAgent 命令

```cmd
EndpointClientAgent --start-module 进程名 卸载密码
```
