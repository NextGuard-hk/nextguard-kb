## 终端白名单

下发路径：

```
C:\Program Files\SkyGuard\SkyGuardEndpoint
```

### 白名单说明

1. 单纯的没有配置file-access=lite的情况，fsmonitor需要配置
2. 配置了file-access=lite，这种可以例外域名白名单
3. 域名只能通过http、https通道去例外

> **注意：** edge浏览器的json文件中默认配置了file-access

## 终端策略文件解密方法

### Windows解密

```bash
cd C:\Program Files\SkyGuard\SkyGuardEndpoint
ucsc_util.exe supper@SG8660 --decrypt_ae
```

### Linux环境解码用des

在Linux环境中使用des解码终端配置文件。
