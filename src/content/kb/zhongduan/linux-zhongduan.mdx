## 终端日志收集命令

```bash
cd /opt/SkyGuard/SkyGuardEndpoint/EndpointAgent
source ./env.sh
EndpointAgentUtil --log-collect
```

日志输出到桌面上。

## 终端 ETS debug

```bash
1. 进入 /opt/SkyGuard/SkyGuard Endpoint/EndpointAgent
2. 执行: source ./env.sh
3. EndpointAgentUtil.exe --module-log-level fsmonitor trace level:
   /opt/SkyGuard/SkyGuardEndpoint/EndpointAgent/var/log
   EndpointFilesystemMonitor TRACE
```

然后取日志：
/opt/SkyGuard/SkyGuardEndpoint/EndpointAgent/var/log 中的日志。
