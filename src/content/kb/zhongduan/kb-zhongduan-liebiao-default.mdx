# KB-终端列表中终端服务器大量出现default字段

## 现象

客户 UCSS 在内网，内网和外网都部署了终端，但是大量终端在终端列表中，终端服务器这一栏显示为 default。

## 原因

虽然创建终端安装包的时候指定了 UCSS 的外网 IP，但是终端连接上报的 DM 的 IP 实际上是 UCSS 的内网 IP，导致获取不到 DM 信息，出现 default 的情况。

## 解决办法

在终端全局设置中配置 UCSS 的外网 IP 即可。

### 配置步骤

1. 进入终端服务器设置页面
2. 在终端服务器列表中找到对应的 UCSS 实例
3. 在“外网IP地址（选填）”字段中填写外部 IP 地址
4. 保证 192 段的网络可达
