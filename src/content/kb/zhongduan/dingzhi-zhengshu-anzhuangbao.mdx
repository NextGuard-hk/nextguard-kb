# 38、定制证书安装包问题

## 验证步骤

### 1. 查看电脑上的证书

安装正常证书生成的安装包后，使用命令行打开证书管理器：

```cmd
certmgr
```

在“受信任的根证书颁发机构”中可以找到 **Endpoint Root CA** 证书。

### 2. 查看证书详情

点开证书查看详细信息：

| 字段 | 值 |
|---|---|
| 有效期 | 2061年7月20日 |
| 使用者 | Endpoint Root CA |
| 公钥 | RSA (2048 Bits) |
| 基本约束 | Subject Type=CA |
| 密钥用法 | Certificate Signing |

### 3. 证书属性说明

- **基本约束**：`Subject Type=CA, Path Length Constraint=None` 表示这是根证书，可以用来颁发子证书
- 当终端访问网站（如 www.baidu.com）时，点击网站证书查看证书路径，可以看到该证书是由上述根证书颁发的

### 4. 安装定制证书包后验证

安装定制证书包后，再次打开证书库，双击新安装的证书进行验证。确认新证书的属性和颁发者信息是否正确。
