# 终端环境如何拦截大的敏感文件及调优

## 常见问题

1. **为什么大的xlsx文件无法拦截**：这是拦截超时的问题，txt文本不需要转换所以不会超时
2. **为什么拦截大文件时应用程序会卡住**：大文件分析需要消耗电脑资源，电脑配置较低时会出现卡顿

## 调优步骤

### 1. 调整阻断模式设置

路径：**系统 → 终端管理 → 全局设置 → 阻断模式**

| 协议 | 超时时间 | 分析大小 |
|---|---|---|
| 终端 HTTP | 15秒 | 12 MB |
| 终端 HTTPS | 15秒 | 12 MB |
| 终端 FTP | 15秒 | 12 MB |
| 终端 SMTP | 35秒 | 100 MB |
| 终端 IM | 300秒 | 100 MB |

### 2. 调整终端资源占用

路径：**系统 → 终端管理 → 全局设置 → 终端资源占用**

- CPU占用上限：100%
- 最低宽带：0 Kbps/秒

## 注意事项

- 根据实际业务需求调整超时时间和分析大小
- 超时时间越长，对用户体验影响越大
- 分析大小越大，消耗的电脑资源越多
