# WIN终端全隐msi安装包制作方法

## 环境要求

- UCSS 3.3 + Win终端 3.5
- 全隐：安装完成强制不弹重启提示框 + 安装重启后默认不显示终端图标
- msi：支持使用第三方工具软件的推送安装

> 注意：此方法没有重启提示框，如果用户安装后不手动重启，则无法拿到最新的策略，可能导致策略不生效。

## 准备工作

### 1.1 配置终端运行模式

在 UCSS 中：系统 → 终端管理 → 终端配置，编辑“默认”配置分组，将终端运行模式设置为“隐性模式”。

### 1.2 收集终端文件

使用一个测试终端属于“默认”终端配置分组，同步后去终端本地收集文件：

路径：`C:\Program Files\SkyGuard\SkyGuard Endpoint\UCSCSDK`

需要收集：
- 隐藏目录 `.ucscsdk\HEAD`
- 配置文件最后一个版本的目录及 `.STATUS` 文件

创建 `UCSCSDK\downloads\profile` 目录，把拷贝出的文件放入。
