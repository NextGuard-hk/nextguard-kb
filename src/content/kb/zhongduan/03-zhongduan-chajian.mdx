# 03.终端插件

## 作用

1. 浏览器上传下载
3. 浏览器打印审计和阻断
2. 浏览器URL水印
3. 百度网盘等通过网页加密传输文件的监控
4. 策略里面如果启用URL为目标，插件抓取不到URL目标

## 常见问题

**1. 终端插件可以自动安装生效吗？**

不可以，插件安装成功后，浏览器都会弹窗提醒，这个是浏览器的功能。

可以通过域控方面，来实现强制安装不提醒。

**临时禁用Office插件方法：**

通过系统 → 终端管理 → 终端应用程序 → 编辑终端应用程序 配置。

参考链接：[WEP - 浏览器插件对功能影响](https://wiki.skyguard.com.cn)
