# 终端审批

## 基本说明

1. 策略动作必须要是阻断，不能是审批，审批是对接第三方的，客户忽略
2. 审批生效的标准：使用相同用户、用户对应的邮件地址、相同通道外发相同文件才行

## 关键配置

- 终端全局配置 > 高级配置 > 开启md5后，才会出现小齿轮编辑，进行手工审批时段设置
- 文件md5是审批的关键指标

## 审批配置文件路径

平台下发到终端的审批配置文件路径：

```
C:\Program Files\SkyGuard\SkyGuardEndpoint\UCSCSDK\var\cache\approval\Approval.db
```

## 定位不生效原因

1. 查看cae文件日志
2. 确认审批配置是否正确下发
3. 检查终端是否正确接收审批策略
