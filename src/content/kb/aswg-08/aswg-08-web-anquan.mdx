# ASWG Web安全网关

## 概述

ASWG (Advanced Secure Web Gateway) 提供Web流量的DLP检测与安全控制功能。

## 主要功能

- HTTP/HTTPS流量检测
- URL过滤与分类
- Web应用控制
- SSL解密检查

## 配置指南

- 代理服务配置
- SSL证书部署
- 访问策略设置

## 常见问题

- HTTPS解密异常处理
- 流量转发性能优化
- 证书信任问题排查
