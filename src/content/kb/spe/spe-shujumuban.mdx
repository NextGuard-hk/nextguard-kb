# 数据模板

## 数据模板类型

逆辑上，是 ucss 从云端下载或者导入最新的模板，然后同步到 spe。

### 预置数据模板后台路径

6c 开头的是预置数据模板：

```
/var/skyguard/sps/program-data/template
```

可将最新的文件拷贝出来给无法联网的 ucss，从页面导入即可。

### 案例

**表格数据误命中**

研发已做以下优化：
- 针对 exec 列之间，匹配手机号码情况，现已忽略制表符这种情况。
- 测试环境参照现场提供的样本，验证通过
- 请导入 v65 的数据模板（EI-10378）

## 可供下载的数据模板

### 0 - 预置模板 v95 & v79

数据模板可以在所有版本上升级，数据分类与策略模板仅支持 3.15sp2 及以上版本。

**注意事项：**

1. 数据模板可以在所有版本上升级
2. 升级存在一定风险，升级前请仔细阅读“更新风险”，确认风险可控后再行升级。
3. 导入策略模板时，将提示“是否覆盖重复规则元素”，一定要选择“否”，否则可能意外引入一些多余的分类组及分类。（Bug EI-12192）

## 预置脚本

预置中国地址脚本，宽模式和窄模式区别：

- **宽模式**：只需要匹配到区
- **窄模式**：需要匹配到街道号

## KB3-topN 前 N 字机密类型数据模板

用于匹配文件前 N 字加密类型的数据模板配置。

## 个人外网邮筱

模版针对个人外网邮筱匹配，包含常见免费邮筱格式识别。

## 文件类型

基于文件类型进行数据匹配的模板配置。

## KB-3.3 版本 ucss 添加脚本内容说明

在 3.3 版本 ucss 中新增的脚本内容类型说明与配置方法。

## cp 数据模板自动生成操作说明

### 操作流程

1. 娱设 CP 数据模板路径
2. 设置计划任务定期生成模板
3. 导入自动生成的模板到 ucss

### CP 数据模板自动生成配置

CP 数据模板自动生成功能可以定期更新机密数据模板，确保模板始终与最新数据保持同步。

## 预置常用脚本类型说明

常用的预置脚本类型包括：

- 身份证号码匹配
- 手机号码匹配
- 中国地址匹配（宽模式/窄模式）
- 个人外网邮筱匹配
- 文件类型匹配
- topN 前 N 字机密类型匹配
