# KB-策略json相关

## 查看当前生效的策略文件内容

```bash
cd /opt/skyguard/ucs/bin
source env.sh
./spe_internal_util -4 engine getPolicyFile
```

`/opt/skyguard/ucs/bin` 路径下会生成 `policy.json` 文件。

## 查看策略版本

```bash
./spe_internal_util -4 engine getPolicyVersion
```
