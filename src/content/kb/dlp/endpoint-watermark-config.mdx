# 終端水印去秒方法並生成安裝包

默認終端水印是顯示時分秒，更新週期是60s，但是秒不會更新，會一直顯示第一次開啟水印的那個秒數。本文將描述如何去秒，並生成一個去秒的安裝包。

## 操作步驟

### 1. 上傳安裝包到UCSS

將主幹版本的安裝包上傳到UCSS服務器，並完成創建。

### 2. 安裝終端

在終端電腦上安裝創建的該安裝包。

### 3. 修改配置文件

進入終端電腦安裝路徑：

```
C:\Program Files\SkyGuard\SkyGuard Endpoint\EndpointAgent\etc
```

拷貝出以下兩個文件：
- `EndpointClientGUIConfig.xml`
- `EndPointWaterMarkConfig.xml`

### 4. 修改TextWaterShowSecond

修改`EndPointWaterMarkConfig.xml`文件，將：

```xml
<TextWaterShowSecond>true</TextWaterShowSecond>
```

修改為：

```xml
<TextWaterShowSecond>false</TextWaterShowSecond>
```

### 5. 重新包裝

將修改後的配置文件放回安裝包，重新生成安裝包即可。

> **注意：** 此方法沒有重啟提示框，如果用戶安裝後不手動進行重啟，也無法拿到最新的策略，此時出現的策略不生效、天空衛士水印等問題都是正常的。
