# 终端浏览器插件

## 插件功能

终端浏览器插件主要提供以下功能：

1. **浏览器上传下载监控** - 监控通过浏览器的文件上传和下载行为
2. **浏览器打印审计和阻断** - 对浏览器打印操作进行审计或阻断
3. **浏览器URL水印** - 在浏览器页面添加水印
4. **加密文件传输监控** - 监控通过网页加密传输的文件

## 插件配置

### 应用程序配置

在终端管理 > 终端应用程序中编辑应用：

- **名称**：如 Watermark-M365
- **类型**：下载应用
- **共享路径/URL**：配置目标URL
- **正则表达式**：匹配规则
- **操作控制**：水印、上传、下载、DLP检测等

### URL目标配置注意事项

- 策略里面如果启用URL为目标，插件可能拦截不到URL目标
- 百度网盘等通过网页加密传输文件需要插件支持

## 浏览器插件常见问题

### 插件未生效

**可能原因**：
- 插件未正确安装
- 浏览器版本不兼容
- 插件被浏览器禁用

**解决方法**：
1. 检查浏览器扩展管理中插件是否已启用
2. 确认浏览器版本满足要求
3. 重新安装插件

### 插件与应用程序文件访问关系

浏览器插件和应用程序的文件访问关系说明：

- 插件主要负责浏览器内的文件操作监控
- 应用程序通道负责本地文件系统的监控
- 两者共同配合实现完整的DLP保护

### 配置终端浏览器打印阻断

配置步骤：

1. 确保浏览器插件已正确安装
2. 在策略中启用浏览器打印控制
3. 配置打印规则（审计/阻断）
4. 测试验证打印控制是否生效
