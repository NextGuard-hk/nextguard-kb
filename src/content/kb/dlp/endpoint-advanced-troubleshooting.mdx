# 终端高级故障排查

## 终端查看方法

使用Process Explorer（procexp）工具检查终端DLL注入状态：

### 检查步骤

1. 下载并运行Process Explorer（来自Sysinternals）
2. 在View菜单中选择**Lower Pane View** > **DLLs**（快捷键Ctrl+D）
3. 选择目标进程（如IE、Chrome等）
4. 在下方DLL列表中查找`sgephook.dll`
5. 如果找到该DLL，表示终端已成功注入（injecting）

> **说明**：sgephook.dll是终端的动态链接库钩子文件，用于监控进程的文件操作行为。

## 终端列表出现default字段

### 问题现象

客户UCSS在内网，内网和外网都部署了终端，但是客户没有大量终端在终端列表中，终端服务器这一栏显示为default。

### 原因分析

虽然创建终端安装包的时候指定了UCSS的外网IP，保证了步交互，但是终端连接上报的DM的IP实际上是UCSS的外网IP，获取不到DM信息，出现default的情况。

### 解决办法

在终端全局设置中配置UCSS的外网IP即可。在终端服务器配置中填入外网IP地址，保证192段网络可达。

## 终端替换证书方法

### 使用createsetup.sh替换证书

```bash
bin/bash ./createsetup.sh -i 192.168.3.200 -p \
  data/ssl/dlp_inter_ca.crt -g /var/skyguard/sps/program-data/setup/ep_install/
  180029267.exe -e /var/skyguard/sps/program-data/267/Tesla.p12:sunny
```

### 参数说明

- `-i`：如果客户使用域名方式生成终端安装包，此处应设置域名
- `-c 7`：禁用终端自我保护，没有该项时使用天空卫士自签名
- `-e`：p12文件位置和密码（格式：`/var/ets/test.p12:1234`）
- 客户自定义证书需要使用2048 bit的密钥

### 注意事项

- `-e`参数指定证书，最后冒号后面是密码
- 替换成客户自己的证书后，可能出现证书不受信任的提示

## 定制证书安装包问题

### 问题场景

客户需要在终端安装包中嵌入自定义SSL证书，用于HTTPS解密检测。

### 操作步骤

1. 准备客户的p12格式证书文件
2. 使用createsetup.sh脚本生成新的安装包
3. 测试新安装包的证书验证功能
4. 部署到目标终端

## 终端环境拦截大文件

### 问题描述

终端环境如何拦截大的敏感文件及调优。

### 解决方案

1. 调整终端策略中的文件大小限制参数
2. 配置分块扫描策略，对大文件进行分段检测
3. 优化内容分析引擎，避免大文件扫描导致系统卡顿
4. 对特定类型的大文件设置白名单

### 调优建议

- 设置合理的文件大小阈值
- 对于超大文件，可以仅扫描文件头部和尾部
- 监控终端性能指标，及时调整策略

## HTTP/HTTPS通道拦截问题

### 问题描述

终端环境下HTTP/HTTPS通道无法拦截文件传输。

### 排查步骤

1. 确认终端浏览器插件已正确安装并启用
2. 检查HTTPS解密证书是否已正确部署
3. 确认策略中是否包含HTTP/HTTPS通道监控规则
4. 检查是否有浏览器版本兼容性问题

### 解决方案

- 重新安装浏览器插件
- 更新终端到最新版本
- 检查并更新SSL证书配置
- 确认防火墙未阻断终端通信

## 终端应用程序不生效

### 问题描述

配置了终端应用程序控制策略，但实际未能生效。

### 排查步骤

1. 确认策略已正确推送到终端
2. 检查应用程序进程名是否正确配置
3. 确认终端服务正常运行
4. 检查是否存在策略优先级冲突

### 常见原因

- 进程名配置错误（大小写敏感）
- 策略未及时同步到终端
- 终端版本不支持该应用程序监控
- 安全软件冲突导致钩子注入失败

## 蓝屏dump日志收集

### 操作目的

通过导入注册表手动触发Windows蓝屏，收集dump日志用于问题排查。

### 操作步骤

1. 导入注册表项启用手动蓝屏触发功能
2. 当问题复现时，通过键盘快捷键触发蓝屏
3. 系统生成dump文件
4. 收集dump文件提交技术支持分析

### 注意事项

- 此操作会导致系统蓝屏重启，**请确保已保存所有工作**
- 仅在技术支持指导下操作
- dump文件默认保存在`C:\Windows\MEMORY.DMP`

## 配置终端浏览器打印阻断

### 功能说明

配置终端环境下的浏览器打印阻断功能，防止用户通过浏览器打印泄露敏感信息。

### 配置步骤

1. 在终端策略中启用浏览器打印监控
2. 配置打印阻断规则
3. 设置敏感内容检测模板
4. 推送策略到目标终端

### 支持的浏览器

- Chrome
- Firefox
- Edge
- IE

## 修改终端常驻图标提示文字

### 问题描述
