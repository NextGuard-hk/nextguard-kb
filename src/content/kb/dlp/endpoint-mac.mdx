# Mac终端部署与管理指南

## 安全模式卸载

### 方法1：手动卸载（已验证）

1. 进入恢复模式
2. 打开Terminal终端工具
3. 删除plist文件

```bash
rm /Volumes/Macintosh HD/Library/LaunchDaemons/com.skyguard.*
rm /Volumes/Macintosh HD/Library/LaunchAgents/com.skyguard.*
```

4. 删除应用文件

```bash
rm -rf /Volumes/Macintosh HD/Library/Application Support/SkyGuard
rm -rf /Volumes/Macintosh HD/User/<用户名>/Library/Application Support/SkyGuard
```

5. 重启

### 方法2：脚本卸载

使用DLP-uninstall卸载脚本：

```bash
bash uninstall_endpoint.sh "卸载密码"
```

## 系统崩溃恢复（Recovery模式）

### 问题描述

安装Mac终端后可能引起系统崩溃，特别是新macOS版本发布后（如10.15），苹果安全机制加固可能导致系统无法正常启动。

### 解决方案

1. **进入Recovery模式**：按住`Command+R`键开机，等待进入恢复界面
   - 如果无法进入Recovery模式，尝试`Option+Command+R`

2. **在Recovery模式下删除终端文件**：

```bash
# 删除LaunchDaemons和LaunchAgents
rm -rf '/Library/LaunchDaemons/com.skyguard.*'
rm -rf '/Library/LaunchAgents/com.skyguard.*'

# 删除主目录
rm -rf /Library/Application\ Support/SkyGuard
```

3. **重启Mac主机**
   - 正常情况下应能进入操作系统
   - 如未能正常启动，检查进程：`name "skyguard.endpoint"`

4. **重启后重新安装终端**（建议自动关闭SIP）
   - 使用默认勾选关闭SIP的安装包
   - 安装后检测终端文件：`ucsc_export_config.xml`

## M1芯片Mac配置

### 关闭SIP和降低安全性步骤

1. **长按电源键**直到出现启动选项界面
2. 点击**Options**进入下一步
3. 从菜单栏点击**Startup Security Utility**
4. 选择**Reduced Security**（降低安全性）
5. 勾选允许第三方内核扩展
6. 重启后在Terminal中确认SIP状态：

```bash
csrutil status
```

## M2芯片终端安装

- M2芯片Mac需要使用专用的MEP-M2安装包
- 安装前需要确认已降低安全性并关闭SIP
- 支持通过MDM（如Micromdm）进行远程部署

## 全隐pkg安装包制作

### 环境要求

- UCSS 3.3 + 终端 3.5版本

### 特性

- 安装完成后**强制不弹重启提示框**
- 安装重启后**默认不显示终端图标**
- 支持通过第三方MDM软件（如Micromdm）推送安装

### 制作步骤

1. 在UCSS中配置终端配置，编辑"默认"配置分组
2. 终端运行模式选择**隐性模式**
3. 启用配置并导出pkg安装包

> **注意**：此方法安装后不弹重启提示框，如果用户安装后不手动重启，UCSS服务器将无法管理该终端，终端也无法获取最新策略。

## Mac卡顿问题处理

有驱动版本的终端，对个别应用使用无驱动版本的配置：

```
networkbypass
file-access=nolite
default-inject=0
js-hook-enable=0
default-file-monitor=1
interaction-monitor=1
is-imapp=1
```

## 已知问题

### GoLand Debug导致黑屏重启

- **环境**：Mac 11.3.1，终端版本3.5.0，SIP为disable
- **现象**：打开GoLand执行Debug后导致黑屏重启
- **解决方案**：终端应用程序添加进程名`debugserver`，设置信任，进程属性设为`allbypass`

### 飞书软件回车键异常

- **现象**：Mac终端安装后，飞书软件回车发送消息变成换行功能
- **状态**：问题已定位，通过日志路径排查

## 常见问题

### 如何确认终端安装成功？

安装后检测终端文件`ucsc_export_config.xml`是否存在。

### Recovery模式无法进入怎么办？

- Intel Mac：尝试`Option+Command+R`组合键
- M1/M2 Mac：长按电源键直到出现选项界面
