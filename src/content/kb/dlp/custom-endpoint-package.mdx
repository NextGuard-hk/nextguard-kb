## 配置文件

custom_config.xml 用于定义源文件和替换文件路径。

## 关闭Office办公软件的注入

### 需要关闭注入的程序

- winword.exe
- powerpnt.exe
- excel.exe

### 操作步骤

1. 在已安装的终端电脑中拷贝EndpointAppDataBaseConfig.json文件

拷贝路径：
```
C:\Program Files\SkyGuard\SkyGuardEndpoint\EndpointAgent\etc
```

2. 编辑EndpointAppDataBaseConfig.json文件，删掉相关应用的配置项

示例配置（winword.exe）：
```json
{
  "name": "winword.exe",
  "PROCVERSION": 1,
  "COMPANY": "",
  "FINGERPRINTS": "",
  "Category": ["EP_APPLICATION_WATERMARK"],
  "isBootInject": 0,
  "HookInjecttype": 4,
  "on-creation-inject-method": 3
}
```

注意：excel.exe没有此字段可不用添加。

3. 编辑压缩包

拷贝压缩包中的custom_config.xml文件到桌面，并进行编辑。

新增配置示例：
```xml
<item>
  <source_path>EndpointAgent/etc/EndpointAppDataBaseConfig.json</source_path>
  <destination_path>%EPAGENT_DIR%\\etc\\EndpointAppDataBaseConfig.json</destination_path>
</item>
```

custom_config.xml完整示例：
```xml
<?xml version="1.0" encoding="utf-8"?>
<items>
  <item>
    <source_path>EndpointAgent/etc/EndpointAppDataBaseConfig.json</source_path>
    <destination_path>%EPAGENT_DIR%\\etc\\EndpointAppDataBaseConfig.json</destination_path>
  </item>
</items>
```
