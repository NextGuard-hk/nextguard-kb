# 终端UUID问题排查

## UUID概述

UUID是终端的唯一标识符，用于在平台上唯一识别每个终端设备。UUID重复会导致多个终端在平台上显示为同一设备。

## UUID重复问题

### 问题现象

多个终端在平台上显示相同的UUID，导致终端管理混乱。

### 原因分析

- 通过镜像克隆的方式部署终端，导致UUID相同
- 终端安装包未正确生成新UUID

### 解决办法 - 重新生成UUID

1. 停止终端保护
2. 删除UUID配置文件
3. 重新启动终端服务，系统会自动生成新的UUID

## 查询终端UUID重复

通过数据库查询终端UUID是否存在重复：

```sql
SELECT uuid, COUNT(*) as count 
FROM endpoint_table 
GROUP BY uuid 
HAVING COUNT(*) > 1;
```

### 操作步骤

1. 编写UUID清理脚本
2. 通过组策略将脚本推送到目标终端

### 注意事项

- 执行清理脚本后终端会重新注册，历史记录可能丢失
- 建议在维护窗口期间执行
- 执行前备份终端配置
