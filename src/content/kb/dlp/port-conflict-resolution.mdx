## 故障现象

终端网络驱动与天擎网络驱动冲突，访问加载网页慢，严重的话还可能导致系统卡死。

## 故障分析

TCP层面端口冲突。

## 解决方案

### 方法1：注册表修改（少量终端）

强制指定代理默认端口为TCP 6553，当只有少量终端需要解决时使用。

命令：
```cmd
reg add "HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\services\sgeptproxy\Parameters" /v "ProxyPort" /t REG_DWORD /d 6553 /f
```

### 方法2：批量修改注册表（大量终端）

当所有终端都需要修改时，需要做一个注册表上的级批量自动修改。

### 方法3：终端包定制

终端包定制步骤：

1. 确认终端版本，提取已部署终端上驱动文件

驱动文件路径：
```
C:\Program Files\SkyGuard\SkyGuard Endpoint\Drivers
```

2. 复制Drivers目录，保留每个文件夹中和代理相关的文件

保留文件：
- sgeptproxy.inf
- sgeptproxy.inf.0
- sgeptproxy.inf.1

3. 用文本编辑工具（如notepad++）修改sgeptproxy.inf文件中的端口配置
