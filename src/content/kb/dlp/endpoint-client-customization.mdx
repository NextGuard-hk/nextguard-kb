# 终端客户端标识自定义配置

## 概述

终端安装后，系统托盘中会显示常驻图标，并通过气泡提醒展示相关信息。可以通过自定义语言模板修改这些显示文字，实现客户端标识（品牌标识）的个性化配置。

## 配置步骤

### 步骤1：进入语言模板设置

导航到 **系统 > 终端管理 > 全局设置** 页面，下拉到语言模板部分。

可以看到两个选项：
- **默认模板**：系统内置模板
- **自定义模板**：用户自定义模板

### 步骤2：下载默认模板

1. 选择"自定义模板"选项
2. 点击"下载默认模板"按钮，下载 `3.8.1_ui_resource.zip` 文件
3. 双击下载的压缩包，到解压缩界面，**但不要解压出来**
4. 将目标文件从压缩包中复制到桌面

### 步骤3：编辑自定义模板

在桌面上找到并编辑 `ui_config.json` 文件，修改以下字段：

```json
{
  "textAlertInstallSame": "已安装了相同版本",
  "textAlertUpdate": "即将升级天空卫士数据泄露防护软件",
  "textAlertUninstall": "确认要进行此操作吗？",
  "textAlertInstallCover": "是否覆盖原有版本",
  "window": {
    "textWinTitle": "春田花花玩具"
  }
}
```

> **说明**：`textWinTitle` 字段即为客户端标识，显示在终端气泡提醒的标题栏中。可以修改为客户的公司名称或产品名称。

### 步骤4：上传自定义模板

1. 将修改后的文件重新压缩（保持原有目录结构）
2. 回到 **系统 > 终端管理 > 全局设置** 的语言模板部分
3. 点击"上传自定义模板"按钮，上传修改后的压缩包
4. 保存当前配置，等待终端同步

## 注意事项

- 压缩包文件中的目录结构必须保持原有格式，不能随意修改
- 上传后需要等待终端从服务器拉取最新配置，可能需要几分钟
- 如果需要恢复默认模板，点击"当前自定义模板"旁边的"默认模板"按钮即可

## 应用场景

- 为客户定制专属的安全软件品牌展示
- 统一企业内部安全软件的显示名称
- 减少终端用户对安全软件品牌的疑虑
