# Mobile DLP

Nextguard **Mobile DLP** extends enterprise DLP policies to mobile devices, protecting sensitive data on smartphones and tablets.

## Supported Platforms

- **iOS** 15+ (iPhone, iPad)
- **Android** 11+ (Samsung, Huawei, Xiaomi, etc.)

## Key Features

- **App-level DLP** - Monitor data sharing between managed apps
- **Copy/Paste Control** - Prevent clipboard leaks from work apps to personal apps
- **Screenshot Protection** - Block screenshots in managed applications
- **Secure Container** - Encrypted workspace for corporate data
- **Remote Wipe** - Erase corporate data on lost/stolen devices

## Deployment

Mobile DLP requires the Nextguard Mobile Agent, deployed via:

1. **MDM Integration** - Push via Microsoft Intune, VMware Workspace ONE, or MobileIron
2. **QR Code Enrollment** - Self-service enrollment for BYOD users
3. **App Store** - Download from Apple App Store or Google Play

## Policy Management

Policies are managed centrally in **UCSS > DLP > Mobile Channel** and support per-app, per-user, and per-device rules.
