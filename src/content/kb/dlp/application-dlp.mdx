# Application DLP

Nextguard **Application DLP** provides visibility and control over sensitive data accessed and transferred through business applications.

## Monitored Applications

- **Instant Messaging** - WeChat Work, Microsoft Teams, Slack, WhatsApp Web
- **Cloud Storage** - OneDrive, Google Drive, Dropbox, Box
- **Webmail** - Gmail, Outlook Web, Yahoo Mail
- **Social Media** - LinkedIn, Facebook, Twitter
- **Database** - SQL queries containing sensitive data
- **Custom Apps** - Any application via API integration

## Detection Capabilities

- Content inspection of file uploads and messages
- Screenshot and screen recording detection
- Clipboard monitoring across applications
- Watermark injection for document tracking

## Policy Configuration

Application DLP policies are managed via **UCSS > DLP > Application Channel**:

1. Select target applications or application categories
2. Define content detection rules
3. Set enforcement actions (Block, Warn, Log, Encrypt)
4. Assign to user groups or departments
