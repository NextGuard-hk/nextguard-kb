# Linux终端

## 支持的操作系统

- **麒麟操作系统** - 国产Linux发行版
- **Ubuntu** - 常见Linux桌面发行版
- 其他主流Linux发行版

## 安装部署

### 安装前准备

1. 确认操作系统版本和架构
2. 确认服务器连接信息
3. 获取对应版本的安装包

### 安装步骤

1. 上传安装包到目标机器
2. 执行安装命令
3. 配置服务器连接信息
4. 验证终端状态

## Linux终端进程属性

```
file-access-open-api=true  # 审计应用程序通道文件拖拽和文件另存为行为
```

## 麒麟操作系统注意事项

- 麒麟系统可能需要特殊的内核模块支持
- 部分麒麟版本的安全机制可能影响终端安装
- 建议安装前确认具体的麒麟版本兼容性

## Ubuntu操作系统注意事项

- 支持Ubuntu标准LTS版本
- 需要确认内核版本兼容性
- 安装后检查服务状态

## 常见问题

### 终端服务无法启动

1. 检查系统日志
2. 确认依赖包是否完整
3. 检查权限设置

### 终端离线

1. 检查网络连接
2. 确认端口连通性（8836端口）
3. 检查防火墙规则
