# Windows终端参考手册

本文档为Windows平台终端DLP管理的完整参考手册，涵盖常用命令、日志路径、进程列表及系统变量。

## 终端日志数据流转

终端监控页面每10分钟去dm取一次数据，因此终端状态更新是10分钟一次。

## 终端策略不生效原因

1. License授权已满，会导致平台策略下发异常
2. `ors.db`没有正常下发，可以更新策略，查看终端电脑 `C:\ProgramData\SkyGuard\` 路径下的文件是否更新；如不更新，检查UCSS中 `orsdb` 下发状态

## Win日志收集

使用以下命令收集调试日志（需以管理员身份运行CMD）：

```bash
ucsc_util.bat --logging_level DEBUG/TRACE/INFO
```

### 步骤

1. 以管理员身份运行CMD
2. 开启debug：
   ```
   cd "C:\Program Files\SkyGuard\SkyGuardEndpoint\EndpointAgent\bin"
   ucsc_util.bat --logging_level DEBUG
   ```
3. 收集日志：
   - 64位系统：`cd "C:\Program Files\SkyGuard\SkyGuardEndpoint\EndpointAgent\bin"`
   - 32位系统：`cd "C:\Program Files (x86)\SkyGuard\SkyGuardEndpoint\EndpointAgent\bin"`
   - 执行：`EndpointAgentUtil.exe --log-collect`
4. 关闭debug：
   ```
   ucsc_util.bat --logging_level INFO
   ```

## 停模块操作

以管理员身份运行CMD，进入目录 `C:\Program Files\SkyGuard\SkyGuardEndpoint\EndpointAgent\bin`

| 操作 | 命令 |
|------|------|
| 停保护 | `EndpointAgentUtil.exe --stop-protect ""` 或 `EndpointAgentUtil.exe --stop-protect abc` |
| 停驱动 | `EndpointAgentUtil.exe --try-safe-stop` |
| 禁用网络模块 | `EndpointAgentUtil.exe --stop-module EndpointTrafficServer` |
| 禁用设备控制模块 | `EndpointAgentUtil.exe --stop-module EndpointFilesystemMonitor` |
| 禁用文件访问 | `EndpointAgentUtil.exe --stop-module EndpointFilesystemMonitor` |
| 卸载终端（未验证） | `EndpointAgentUtil.exe --uninstall-endpoint` |
| 退出终端 | `EndpointWatchdog --quit/stop/exit` |
| 启动终端 | `EndpointWatchdog --start` 或 `EndpointClientAgent --start` |

### 查看模块状态

## 终端主服务管理

```bash
sc query sgepsvc      # 查看状态
sc stop sgepsvc       # 停止服务
sc start sgepsvc      # 启动服务
```

## 主要日志路径

| 日志类型 | 路径 |
|----------|------|
| 安装日志 | `%temp%` 目录下 `EndpointInstaller.log` & `EndpointInstaller_XXX.log` |
| 策略匹配结果/敏感文件超时漏拦截 | `\UCSCSDK\log\ContentAnalysisEngine.log` |
| 协议分析耗时timeout | `\UCSCSDK\var\log\spe_access.log` （total time elpase(ms)检测总耗时，包含内容提取、matching应的时间是策略匹配） |
| 注册失败/终端配置及策略更新/AD同步失败 | `\UCSCSDK\var\log\ucsc_sdk.log` |
| 数据发现/数据扫描 | `\EndpointAgent\var\log\EndpointDataDiscovery.log` |
| 终端配置及策略更新 | `\EndpointAgent\log\EndpointClient.log` |
| 网络问题 | `\SkyGuard Endpoint\EndpointAgent\var\log\EndpointTrafficServer.log` |
| dump/蓝屏日志 | `%SystemRoot%\MEMORY.DMP` 及 `C:\Program Files\SkyGuard\SkyGuard Endpoint\xxx` |

## 终端停护恢复

终端停护后如何恢复：

1. 手动终结 Endpoint Client Agent 进程，watchdog 会拉起来，保护会再次打开
2. `EndpointWatchdog --start` 或 `EndpointWatchdog --quit`
3. 或者直接用 `EndpointClientAgent --start-module 进程名 卸载密码`

## 终端忘记密码卸载方式

进入安全模式，执行脚本：`safemode_uninstall_wep.bat`
