## 01、定制终端安装包关闭Web过滤功能

### 操作步骤

1. 找一台安装了agent的电脑，找到文件ucsc_sdk_config并解密
2. 修改配置：将WFELite.exe的is_start设置为false

```xml
<start_cmd>WFELite.exe</start_cmd>
<is_start>false</is_start>
```

3. 解密命令：

```cmd
cd "C:\Program Files\SkyGuard\SkyGuard Endpoint\UCSCSDK\bin"
ucsc_util.exe --decrypt des "C:\custom_config.xml" C:\
custom_config1.xml
```

4. 修改完成后重新加密：

```cmd
ucsc_util.exe --encrypt des "C:\custom_config1.xml" C:\
custom_config.xml
```

5. 替换后放入安装包并重新打包
