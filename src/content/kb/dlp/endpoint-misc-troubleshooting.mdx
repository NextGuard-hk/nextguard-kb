# 终端杂项故障排查指南

## 关闭自动云端更新URL

### 问题描述

安装终端后发现电脑访问不明地址9.4.11.176（bedp.skyguard.com.cn），这是URL库更新地址，默认开启。

### 解决方案

修改配置文件关闭WFELite.exe：

1. 找到安装了agent的电脑，定位文件`ucsc_sdk_config`
2. 解密并修改配置
3. 将WFELite.exe的`is_start`设为`false`

```xml
<component name="WFELite">
  <start_cmd>WFELite.exe</start_cmd>
  <is_start>false</is_start>
</component>
```

### 解密命令

```bash
cd "C:\Program Files\SkyGuard\SkyGuard Endpoint Agent\bin"
ucsc_tool.exe -d ucsc_sdk_config ucsc_sdk_config.xml
```

## 禁用WPS同步功能

WPS同步进程可能导致数据泄露，需要通过终端管控禁止。

### WPS同步相关进程

- `wpscloudlaunch.exe`
- `wpscloudsvr.exe`

### 操作步骤

1. 在终端管理中新建两个应用程序
2. 分别添加进程名`wpscloudlaunch.exe`和`wpscloudsvr.exe`
3. 设置进程属性为**block**

> **注意**：程序名称只能通过选择获取，不能手动输入。若不知道程序名称所在位置，可在任务管理器详细信息中查找。

## 取消explorer.exe驱动注入

### 问题描述

终端安装后可能导致开机黑屏问题，通过取消explorer.exe的驱动注入可解决，同时保证网络共享、移动存储、本地存储通道管控仍然有效。

### 解决方案

1. 打开终端安装压缩包（不要解压）
2. 进入`custom\EndpointAgent\etc`目录
3. 复制`EndpointClientAgentConfig.xml`文件到桌面
4. 编辑该文件，删除explorer.exe相关的驱动注入字段
5. 将修改后的文件放回压缩包中
6. 重新安装终端

## 微信接收图片监控

### 问题描述

微信接收的图片文件需要被终端DLP监控和审计。

### 配置方法

确保终端策略中包含对微信应用程序的监控规则，特别是图片文件传输通道。

## 终端电脑内存占用高

### 问题描述

终端进程占用过多内存。

### 解决方案

1. 检查终端版本是否为最新版
2. 检查策略配置是否合理
3. 考虑调整进程监控范围
4. 如持续异常，收集日志提交技术支持

## 相互加白路径配置

当终端与其他安全软件冲突时，需要配置相互加白：

1. 在终端中添加其他安全软件的进程到白名单
2. 在其他安全软件中添加终端进程到信任名单
3. 终端安装目录默认：`C:\Program Files\SkyGuard\`

## 终端环境检查

安装终端前建议检查以下环境要求：

- 操作系统版本兼容性
- 磁盘空间是否充足
- 是否存在其他安全软件冲突
- 网络连接状态
- 管理员权限

## 终端静默安装

支持通过命令行进行静默安装，适用于批量部署场景。

## 终端应用程序管理

### HTTP截断配置

配置终端浏览器环境下的HTTP截断功能，用于监控网页上传下载行为。

### 注册表导入

通过导入注册表手动修改终端配置：

1. 导出当前注册表配置作为备份
2. 编辑注册表文件修改目标配置
3. 导入修改后的注册表文件
4. 重启终端服务使配置生效

## 终端大量离线问题

### 可能原因

1. 服务器端连接异常
2. 网络策略变更导致通信中断
3. 终端版本与服务器版本不兼容
4. 防火墙或代理服务器阻断了终端通信

### 排查步骤

1. 检查网络连接状态
2. 确认服务器地址配置正确
3. 检查防火墙规则
4. 查看终端日志中的错误信息

## UCSS版本兼容性

### 3.3版本UCSS注意事项

- 确认终端版本与UCSS版本兼容
- 检查license授权数量是否充足
- UCSS创建终端配置时注意版本匹配

## 终端停护处理

终端停护后如何恢复保护：

1. 检查终端服务状态
2. 重启终端服务
3. 如无法恢复，重新安装终端
4. 检查策略配置是否正确推送
