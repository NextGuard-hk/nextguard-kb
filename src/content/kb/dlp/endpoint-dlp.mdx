# Endpoint DLP

Nextguard **Endpoint DLP** provides data-at-rest and data-in-use protection directly on user workstations and laptops.

## Protected Channels

- **USB / Removable Media** - Block or audit file transfers to USB drives
- **Print** - Prevent printing of sensitive documents
- **Clipboard** - Control copy/paste of sensitive content
- **Screen Capture** - Block screenshots of protected applications
- **Application Control** - Restrict file access by unauthorized applications
- **Cloud Sync** - Monitor uploads to cloud storage (OneDrive, Google Drive, Dropbox)

## Agent Deployment

The Endpoint DLP agent is deployed via:

1. **UCSS Push Install** - Remote deployment from UCSS console
2. **MSI Package** - Distribute via GPO, SCCM, or Intune
3. **Manual Install** - Direct installation on individual endpoints

## Supported Platforms

| OS | Version | Architecture |
|----|---------|-------------|
| Windows | 10 / 11 / Server 2016+ | x64 |
| macOS | 12 Monterey+ | ARM64 / x64 |
| Linux | Ubuntu 20.04+, RHEL 8+ | x64 |

## Policy Enforcement

Endpoint DLP policies support:

- **Content-aware rules** - Inspect file content before allowing transfers
- **Device control** - Whitelist/blacklist specific USB devices by vendor ID
- **User notification** - Show popup warnings when policy is triggered
- **Offline enforcement** - Policies remain active even when disconnected from network
