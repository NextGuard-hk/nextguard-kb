## 终端白名单

### 白名单文件位置

白名单配置文件下发路径：

```
C:\Program Files\SkyGuard\SkyGuardEndpoint\UCSCSDK
```

### 配置说明

1. **无file-access=lite配置：** fsmonitor无法匹配域名白名单
2. **有file-access=lite配置：** 可以例外域名白名单，但注意 Edge浏览器的JSON文件默认配置了file-access=lite
3. **UNC目标域名/网络共享：** 目标IP白名单可以在fsmonitor中配置
4. **域名限制：** 域名只能通过HTTP、HTTPS通道去例外

## 终端策略文件解密

### Windows环境

```bash
cd C:\Program Files\SkyGuard\SkyGuardEndpoint\UCSCSDK\bin
ucsc_util.exe supper@SG8660 --decrypt aes C:\path\to\config.json
```

解密成功后显示：

```
decrypt file success, C:\ep\ep_profile -> C:\ep\ep_profile.dec
TT_SOCKET 53560
```

### Linux环境

```bash
./ucsc_util supper@SG8660 --decrypt aes /path/to/config.json
```

解密后可以查看策略配置文件的内容，用于排查策略下发是否正确。

## 常见问题

### 截屏保存不生效

终端截屏功能不生效时，检查：

- 终端策略是否已下发
- 截屏功能是否在策略中启用
- 终端客户端是否正常运行

### 与其他安全软件冲突

如与天擎或亿赛通等产品冲突，需要：

- 在第三方安全软件中添加SkyGuard终端进程白名单
- 在SkyGuard终端中添加第三方安全软件白名单
- 确认双方的驱动程序不会互相拦截
