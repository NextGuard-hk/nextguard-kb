# WIN終端全隱MSI安裝包製作

## 概述

- **環境：** 3.3UCSS + 3.5Win終端
- **全隱：** 安裝完成強制不彈重啟提示框 + 安裝重啟後默認不顯示終端圖標
- **MSI：** 支持使用第三方工具軟件的推送安裝

> **注意：** 此方法沒有重啟提示框，如果用戶安裝後不手動進行重啟，也無法拿到最新的策略，此時出現的策略不生效、天空衛士水印等問題都是正常的。

## 準備工作

### 1.1 獲取安裝重啟後默認不顯示終端圖標所需配置

進入 **系統 → 終端管理 → 終端配置**，編輯"默認"配置分組，終端運行模式選擇「隱性模式」。

### 1.2 獲取安裝完成強制不彈重啟提示框所需配置

在終端配置中的「安裝模式」選擇静默安裝。

## 操作步驟

### 步驟1：下載配置文件

在UCSS上配置好終端分組後，先安裝一個普通終端，然後從安裝目錄中提取配置文件。

### 步驟2：將配置套用到MSI安裝包

將提取的配置文件套用到MSI安裝包的對應目錄中。

### 步驟3：部署MSI安裝包

使用第三方工具軟件將MSI安裝包推送到目標終端電腦。
