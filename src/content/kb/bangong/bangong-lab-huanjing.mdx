## linux搭建防火墙

### 1、配置网卡及路由信息

2张网卡，一个内网，一个办公网即可

```
vi /etc/network/interfaces

# The loopback network interface
auto lo eth0 eth1
iface lo inet loopback

# The primary network interface
allow-hotplug eth0
iface eth0 inet static
address 172.22.80.243
netmask 255.255.0.0
gateway 172.22.0.1
dns-nameserver 172.21.23.21

allow-hotplug eth1
iface eth1 inet static
address 192.168.0.1
netmask 255.255.255.0
```

验证路由信息:

```
route -n
Kernel IP routing table
Destination     Gateway         Genmask
0.0.0.0         172.22.0.1      0.0.0.0
172.22.0.0      0.0.0.0         255.255.0.0
192.168.0.0     0.0.0.0         255.255.255.0
192.168.1.0     192.168.0.254   255.255.255.0
192.168.2.0     192.168.0.254   255.255.255.0
192.168.3.0     192.168.0.254   255.255.255.0
192.168.4.0     192.168.0.254   255.255.255.0
```

### 2、配置端口映射，保证办公网段可以访问

麒麟系统为例:

```
执行iptables -t nat -A POSTROUTING -o ens33...
iptables-restore < /etc/sysconfig/iptables
```

其中下述需要禁用:
```
#-A INPUT -j REJECT --reject-with icmp-host...
#-A FORWARD -j REJECT --reject-with icmp-ho...
```

配置端口映射规则:
```
vi /etc/iptables/rules.v4
-A PREROUTING -d 172.22.80.243/32 -p tcp...
iptables-restore < /etc/iptables/rules.v4
```

### 3、配置IP转发

```
cat /etc/sysctl.conf
net.ipv4.ip_forward=1
```

## Linux系统搭建路由器

### 1、建议配置4-5块网卡

- eth0接防火墙: 管理口用
- eth1: mta口
- eth2: p1口
- eth3: p2口
- eth4: pc

```
vi /etc/network/interfaces
allow-hotplug eth0
iface eth0 inet static
  address 192.168.0.254
  netmask 255.255.255.0
  network 192.168.0.0
  broadcast 192.168.0.255
  gateway 192.168.0.1
  dns-nameservers 172.21.23.21

allow-hotplug eth1
iface eth1 inet static
  address 192.168.1.254
  netmask 255.255.255.0

allow-hotplug eth2
iface eth2 inet static
  address 192.168.2.254
  netmask 255.255.255.0

allow-hotplug eth3
iface eth3 inet static
  address 192.168.3.254
  netmask 255.255.255.0

allow-hotplug eth4
iface eth4 inet static
  address 192.168.4.254
  netmask 255.255.255.0
```
