# APP服务-Apache与Nginx配置

## 概述

APP模块包含Apache和Nginx两类Web服务器组件，用于提供应用层反向代理和服务托管。

## Apache配置

### 服务管理

```bash
# 启动Apache服务
systemctl start httpd

# 查看状态
systemctl status httpd

# 重启服务
systemctl restart httpd
```

### 配置文件路径

```
/etc/httpd/conf/httpd.conf
/etc/httpd/conf.d/
```

## Nginx配置

### 服务管理

```bash
# 启动Nginx
systemctl start nginx

# 重新加载配置
nginx -s reload

# 测试配置文件
nginx -t
```

### 配置文件路径

```
/etc/nginx/nginx.conf
/etc/nginx/conf.d/
```
