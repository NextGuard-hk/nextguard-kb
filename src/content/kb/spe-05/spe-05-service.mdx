# SPE服务管理

## spe-service服务

spe-service包含的子线程如下：

```bash
ps -auxf
```

主要组件：
- MgmtConsole
- OCRServer
- DeviceManager
- fpdbservice
- ContentAnalysisEngine
- DSA
- LabelManagementService
- XIDServer

## 查看SPE容器主进程状态

```bash
root@UCSS-V3:/etc/init.d#
supervisorctl status spe-service
```

正常状态应显示：RUNNING

## 相关子模块

- **策略**: DLP策略配置与管理
- **KV**: 键值对存储服务
- **DM**: 设备管理服务
- **DSA**: 数据安全审计
- **ICAP**: ICAP协议集成
- **数据模板**: 内容检测模板
- **OCR**: 光学字符识别
- **LMS**: 标签管理服务
