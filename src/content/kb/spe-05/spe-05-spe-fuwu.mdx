# SPE服务管理

## spe-service

包含的子线程如下：

```bash
ps -auxf
/opt/skyguard/ucs/bin/spe-service -p /var/run/spe-service.pid
MgmtConsole
--OCRServer
----OCRServer --slave 1
----OCRServer --slave 0
--DeviceManager
--fpdbservice
--ContentAnalysisEngine
--DSA
--
LabelManagementService
--XIDServer (3.3)
```

## 查看SPE容器主进程状态

```bash
root@UCSS-V3:/etc/init.d#
supervisorctl status spe-service
spe-service    RUNNING    pid 61, uptime 56 days, 23:26:33
```

DM即DeviceManager，主要负责终端接入、数据传输等。
