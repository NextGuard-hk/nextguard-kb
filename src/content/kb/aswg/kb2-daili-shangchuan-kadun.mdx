# KB2-代理模式上传文件卡顿如何定位

## 问题描述

**工单号：** TS-11408

**环境：** ASWG 3.15 正向代理

**问题：** 访问 https://bifrost.nahefa.com.cn ，登入后上传文件，超过100M的文件，进度会一直卡住。

## 排查步骤

1. 关闭代理后，相同文件12分钟左右完成上传
2. 添加 `bifrost.nahefa.com.cn` 的域名SSL例外，网站证书变了，但是仍然卡进度

## 分析方法

在ASWG的ats容器里抓包：

```bash
tcpdump -i eth2 host x.x.x.x or bifrost.nahefa.com.cn -w /opt/skyguard/download/tmp.cap
```

> `x.x.x.x` 代表PC的IP地址

另外，在PC上也需要同时抓包。

抓包后的两个文件发给技术支持进行分析。
