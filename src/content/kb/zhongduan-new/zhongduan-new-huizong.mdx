# 终端new汇总

## 终端白名单

下发路径：
```
C:\Program Files\SkyGuard\SkyGuardEndpoint
```

### 白名单配置说明

1. 单纯的没有配置file-access=lite的情况，fsmonitor会监控文件操作
2. 配置了file-access=lite，这种可以例外域名白名单
3. edge浏览器的json文件种默认配置了file-access
4. UNC的目标域名/网络共享、目标ip白名单可以在file中配置
5. 域名只能通过http、https通道去例外

## 终端策略文件解密

### Windows环境

```bash
cd C:\Program Files\SkyGuard\SkyGuardEndpoint\UCSCSDK\bin
ucsc_util.exe supper@SG8660 --decrypt_ae
```

### Linux环境

```bash
./ucsc_util supper@SG8660
```
