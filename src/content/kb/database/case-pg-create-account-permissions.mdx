# 案例5、pg数据库创建账号，开放终端列表监控权限和表

## 背景

WEP终端：IM通讯工具管控。由于产品的数据库暴露给用户有很大未知风险，而终端的一些信息目前只能通过数据库查询来实现。因此需要创建只读权限的用户，提供给客户使用。

> 注意：本章节内容需要天空卫士同事在 UCSS 上来操作

## 步骤

### 1. 为 postgres 超级账户设置密码

```bash
# root用户登录
passwd postgres
# 密码更改为 firewall

# 切换到 postgres 账户
su postgres

# 进入数据库命令行
psql

# 修改数据库密码
ALTER USER postgres WITH PASSWORD 'postgres';
\q
```

### 2. 创建本地和数据库指定权限用户 ep_visitor

```bash
# 创建本地用户
useradd ep_visitor
passwd ep_visitor

# 连接 postgres 数据库
su postgres
psql sps_db
```

```sql
-- 创建数据库用户并赋权
create user ep_visitor password '123456';
grant usage on schema system to ep_visitor;
grant select on system.ep_monitoring_result to ep_visitor;
grant connect on database sps_db to ep_visitor;
\q
```

### 3. 测试新创建的用户

```bash
su ep_visitor
psql sps_db
```

### 4. 修改密码

```sql
ALTER user ep_visitor with password '2020@sec_dlpYw!';
```

### 5. 赋予视图访问权限

在前一节赋权步骤基础上，添加访问 `system.view_endpoint_computer` 视图的权限：

```sql
grant select on system.view_endpoint_computer to ep_visitor;
```
