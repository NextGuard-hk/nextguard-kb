## 招行2千万海量邮件日志导出方法

### 1、设置1次请求1万条邮件日志

- `"size":10000`
- 设置滚动标识有效期60分钟：`scroll=1m`

```bash
curl -H 'Content-Type: application/json' -XPOST "http://127.0.0.1:9200/email-message*/_search?pretty&scroll=1m" -d '{"size":10000,"query":{"match_all":{}}}'>1
```

### 2、获取上一次的scroll_id，继续查询

```bash
curl -H 'Content-Type: application/json' -XPOST "http://127.0.0.1:9200/email-message*/_search?pretty&scroll" -d '{"scroll": "1m","scroll_id": "your_scroll_id"}'>2
```
