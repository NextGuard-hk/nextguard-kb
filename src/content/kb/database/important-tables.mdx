# 重要的数据库表

## sys_account

查看管理员、用户有效期等信息，修改admin密码过期时间。

```sql
select * from sys_account where id=1;

su postgres
psql -d sps_db
update sys_account set password_expired_time='2025-12-31';
select * from public.sys_account;
```

## system.device_info

设备注册信息。

```sql
select * from system.device_info;
```

正常ucss的device_ip和masterip是同一个ip地址，如果不一致可能是寻找ucss的旧ip，这个master信息只会在导入授权的时候变化。

解决办法：重新导入授权

## system.syslog

系统日志，默认保存3个月。

如果需要修改到更长时间，可以修改配置文件：

```bash
vi /var/skyguard/sps/program-data/config/dlp_*.properties
# 找到 syslog 配置
auto_del_syslog_month=3
```

## 策略相关表的创建者修改

修改各表的creator字段为指定用户（如从 SKYGUARD_ERROR 改为 zwcy）：

```sql
-- 1、策略: plc_policies
update plc_policies set creator='zwcy' where creator='SKYGUARD_ERROR';

-- 2、策略元素-策略动作: plc_actions
update plc_actions set creator='zwcy' where name='邮件工作流';

-- 3、字典/文件类型组/正则表达式: plc_element_set
update plc_element set creater='zwcy' where name='xxx';
update plc_element set creater='zwcy' where name='ELF Executable';

-- 4、指纹: plc_fingerprint
update plc_fingerprint set creator='zwcy' where creator='SKYGUARD_ERROR';

-- 5、基本设置-邮件告警: conf_email_alert
update conf_email_alert set creator='zwcy' where name='邮件告警测试';

-- 6、基本设置-syslog/SIM
update syslog_setting set creator='zwcy' where name='邮件日志测试';

-- 7、用户管理-分级对象: sys_hierarchic
update sys_hierarchic set creator='zwcy' where name='endpoint';

-- 8、账户管理-管理员: sys_account
set search_path to public;  -- 切换路径
update sys_account set creator='zwcy' where login_name='admin';

-- 9、归档设置、归档空间大小对应的数据库表
system.archive_config  -- 归档设置读取失败时可以查询该表
```
