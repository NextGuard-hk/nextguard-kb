## 连接方式

### 方式1：本地连接

```bash
su postgres
psql -d sps_db   #进入业务数据库
psql -d postgres  #进入系统数据库
```

### 方式2：远程连接

```bash
psql -h 127.0.0.1 -U sps sps_db
psql -h 169.254.253.4 -U sps sps_db
```

## PG/ES手动备份恢复

### 备份

```bash
pg_dump -U sps sps_db > /tmp/sps_db_backup.sql
```

### 恢复

```bash
psql -U sps sps_db < /tmp/sps_db_backup.sql
```

## 数据库连接异常排查

当数据库连接异常时，检查以下几点：

1. 确认PG容器是否正常运行
2. 检查端口5432是否监听
3. 查看 `/var/log/postgresql` 日志
4. 检查磁盘空间是否充足
