# Elasticsearch数据库概览

## 常用路径

```
/var/lib/elasticsearch              # 数据库路径
/var/skyguard/es/data               # 3.15
/var/lib/elasticsearch/nodes/0/indices  # 数据库路径
/var/log/elasticsearch/elasticsearch.log  # 日志
/etc/elasticsearch/                  # 系统配置路径
```

## 账号密码

| 版本 | 账号/密码 |
|------|----------|
| 3.9版本 | elastic / elastic |
| 3.10版本 | skyguard / Elastic@SkyGuard253.5 |
| 3.3db高可用 | elastic / elastic |
| 3.10db高可用 | skyguard / Elastic@SkyGuard |

## 常用端口

- 9200、9300 (3.3高可用)

## 配置文件

```
/etc/elasticsearch/jvm.options        # 调整ES内存
-Xms8192M
-Xmx8192M

/etc/elasticsearch/elasticsearch.yml  # 其他配置
```

## 常用参数说明

```
/_search?pretty      # pretty表示用json格式排序
/_search?scroll=1m   # scroll表示滚动查询
/_search?pretty&scroll=1m  # 用&组合pretty和scroll
```

## 常用命令

备注：
- `-H 'Content-Type:application/json'`: 表示用json格式
- `|json_pp`: 序列化显示结果
- 默认是 `http://`

### 查询ES版本信息

```bash
curl -XGET localhost:9200
```

### 查看数据库健康度

yellow: 警告，但是可以使用; red: 不可用

```bash
curl 169.254.253.5:9200/_cat/health?v -u skyguard:Elastic@SkyGuard253.5
```

### 查看事件和日志的索引和大小

```bash
curl 169.254.253.5:9200/_cat/indices -u skyguard:Elastic@SkyGuard253.5
```

### 删除指定网络事件的索引

```bash
curl -s -XDELETE http://127.0.0.1:9200/dlp-network-*
```

### 根据网络事件ID查询网络事件

```bash
curl -X GET "http://169.254.253.5:9200/dlp-network-*/_search?pretty" \
  -u skyguard:Elastic@SkyGuard253.5 |json_pp
```

如果发现乱码，在命令后面加上`&pretty`:

```bash
curl -X GET "http://169.254.253.5:9200/dlp-network-*/_search?pretty" \
  -u skyguard:Elastic@SkyGuard253.5
```

### 根据uuid查询网络事件

```bash
curl -X GET "http://169.254.253.5:9200/dlp-network-*/_search?pretty" \
  -u skyguard:Elastic@SkyGuard253.5 |json_pp
```

### 根据message_id查询网络事件

```bash
curl -X GET 'http://169.254.253.5:9200/dlp-network-*/_search?pretty' \
  -u skyguard:Elastic@SkyGuard253.5
```

### 根据serialId查询对应的messageId

```bash
curl -X GET 'http://169.254.253.5:9200/dlp-network-*/_search?pretty' \
  -u skyguard:Elastic@SkyGuard253.5 |json_pp |grep messageId
```

### 根据索引内指定时段的网络事件

```bash
curl -H 'Content-Type: application/json' -XPOST \
  incident/_search?pretty \
  -d '{"query":{"bool":{"must":[{"range":{"incidentDateTime":{"gte":"2021-06-25T16:00:00.000+0000","to":"2021-06-28T15:59:59.999+0000","include_lower":true,"include_upper":true}}}]}}}'
```

备注：
- `dlp-network-20210527-01` 替换当前环境的index；归档后缀为 `20210527-01`
- `incidentDateTime`: 时间，from:开始时间 to:结束时间
- 查询当前时段之前13分钟的事件

### 查询shell脚本示例

```bash
#!/bin/sh
filename=/testucss/maildlp_`date +%Y-%m-%d`
curl -H "Content-Type: application/json" \
  -XPOST 'http://169.254.253.5:9200/dlp-network-*/_search?pretty' \
  -u skyguard:Elastic@SkyGuard253.5 \
  -d '{"_source":"serialId,transactionId,detectDate,matchedPolicies.uuid,matchedPolicies.name","query":{"bool":{"should":[{"range":{"incidentDateTime":{"gte":"now-13m","lt":"now","format":"yyyy-MM-dd HH:mm:ss","time_zone":"+08:00"}}}]}}}' \
  >> $filename
echo "\n" >> $filename
```

## ES数据库中单条事件大小统计

### DLP事件
- 网络事件、终端事件、移动事件：范围从0.5k-7k不等
- 数据发现事件：每条大约占用3k-7k左右，基本在7K以内
- 以1G大小为例，每条事件以平均5k计算，每种类型可以存储约20万条记录

### SWG日志
- 每条日志在0.5k左右
- 以1G大小为例，可存储200万条记录

### ESG邮件日志
- 单条日志大概在0.6-0.8k，算下来1G硬盘，大约可以存储125万-170万条
- 以上是1-2个收件人的情况，收件人越多，单条日志占用越大

## 查看ES数据库中对应的表大小

```bash
curl http://localhost:9200/_cat/indices/dlp-network-*?v
```

示例输出：
```
green open dlp-network-20221104-01 wQtMBaeEQC...
green open dlp-network-20220802-01 rgjqxRasQT...
green open dlp-network-20220210-01 ouk-gg7LSM...
```

页面备份的日志或者证据文件经过压缩会小于这个数，结合招标文件信息：
- 页面显示网络事件条数：258W
- 内发平台ES数据库统计：网络事件：62G
- 页面备份网络事件大小：37G (压缩比: 43.9/62=0.7)

## 案例：查询网络事件是否有重复id

```bash
curl -H 'Content-Type: application/json' -X POST \
  -u skyguard:Elastic@SkyGuard253.5 \
  -d '{"size":0,"aggs":{"duplicate_ids":{"terms":{"field":"serialId","min_doc_count": 2}}}}'
```

## 海量数据多次查询

使用 Scroll API，你可以在大型数据集上分页查询数据。

### 开始滚动搜索

```json
POST /your_index/_search?scroll=1m {"size":1000, "query":{"match_all":{}}}
```

### 获取每个批次的数据

```json
POST /_search/scroll {"scroll":"1m","scroll_id":"your_scroll_id"}
```

### 清理滚动上下文

当你完成查询时，记得清理滚动上下文：

```json
DELETE /_search/scroll {"scroll_id":["your_scroll_id"]}
```

## 把事件导出为json文件

```bash
curl -H 'Content-Type: application/json' -XPOST \
  'http://169.254.253.5:9200/dlp-network-*/_search?pretty' \
  -u skyguard:Elastic@SkyGuard253.5 \
  -d '{"query":{"term":{"serialId":{"value":625230}}}}' \
  > /tmp/625230.json
```
