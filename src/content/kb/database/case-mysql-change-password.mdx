## 案例：修改mysql密码方法

修改密码方法（建行专版处理方案）：

### (1) 进入后台，登录mysql，修改用户密码

假设新密码为 `new@password`

登录mysql：

```bash
mysql -u cert_server -pcert_server
```

修改用户密码：

```sql
set password for cert_server@localhost=password("new@password");
```

### (2) 修改文件utils.php

文件路径：`/opt/skyguard/www/ats/utils.php` 大概 215行

将 `$mysql_password = "cert_server"` 修改为 `$mysql_password = "new@password"`

### (3) 修改脚本backup.sh

文件路径：`/opt/skyguard/www/ats/backup.sh` 大概 15行

将 `dbpasswd="cert_server"` 修改为 `dbpasswd="new@password"`

### (4) 杀掉进程daemon.php

先找到daemon.php进程 pid：

```bash
ps -ef | grep daemon.php
```

然后执行：

```bash
kill -9 $pid
```

### 验证

使用初始密码登录mysql会报错 `ERROR 1045 (28000): Access denied`，使用更新密码登录即可成功。
