# 修改指纹相似度字段，将最小阈值减低到10%以下

## 背景

目前最小设置阈值时10%，现场需要识别精度最小支持1%，需要通过数据库修改。

## 操作步骤

### 1. 获取策略中文件指纹的condition uuid

页面F12打开开发者工具，获取策略中文件指纹的condition uuid。

### 2. PG数据库设置相似度5%

```sql
sps_db=> update system.plc_rule_conditions set threshold=0.05
where uuid='<condition_uuid>';
UPDATE 1
```

发现修改后，并未修改成功，依旧是0.1。

### 3. 查询表字段精度

threshold字段最小精度为小数点1位，查询表字段信息：

```sql
SELECT table_name, column_name, data_type, numeric_precision, numeric_scale
FROM information_schema.columns
WHERE table_name = 'plc_rule_conditions' AND column_name = 'threshold';
```

结果显示threshold字段类型为numeric，精度2位，小数位1位，因此无法设置小于0.1的值。

### 4. 修改字段精度

需要先修改字段精度，再设置阈值值。
