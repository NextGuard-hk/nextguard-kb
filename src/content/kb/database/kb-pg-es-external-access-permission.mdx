# KB-开启PG、ES外部访问权限

## 操作方法

打开 `/etc/appliance/compose.yaml`，增加红框内的内容：

### PG数据库配置

在pgsql容器配置中增加ports映射：

```yaml
pgsql:
  image: "pgsql"
  restart: always
  networks:
    internal:
      ipv4_address: 169.254.253.4
  ports:
    - 5432:5432
  volumes:
    - /var/skyguard/pgsql/data:/opt/data
    - /var/log/skyguard/pgsql:/opt/log
  healthcheck:
    test: nc -z localhost 5432
    interval: 30s
    timeout: 10s
    retries: 3
    start_period: 30s
```

如果不能连接的话，再执行命令：`ufw allow 5432/tcp`

### ES数据库配置

在es容器配置中增加ports映射：

```yaml
es:
  image: "elasticsearch"
  restart: always
  networks:
    internal:
      ipv4_address: 169.254.253.5
  ports:
    - 9300:9300
    - 9200:9200
  privileged: true
  environment:
    - bootstrap.memory_lock=true
  ulimits:
    memlock:
      soft: -1
      hard: -1
  volumes:
    - /etc/appliance/:/opt/skyguard/www
    - /var/skyguard/es/data:/opt/data
    - /var/log/skyguard/es/:/opt/log
    - /var/skyguard/sps/elasticsearch/backup
  healthcheck:
    test: curl --silent --fail --user skyguard
    interval: 30s
    timeout: 10s
    retries: 3
    start_period: 50s
```

### 生效方法

都改完之后，执行：

```bash
docker-compose -f /etc/appliance/compose.yaml up -d
```

容器就会更新。
