## KB-可视化服务

### 1.架构图

可视化服务、第三方服务平台通过请求Restful API获取所需数据。架构包含 Gateway、DSAG、SWG、Data-Provider 组件，底层连接 Redis、PG、ES 数据库。

### 2.概要

可视化服务、第三方服务平台通过请求Restful API获取所需数据。

在访问API之前，需要进行认证，通过请求Check-in接口并在header加入Authorization:Basic base64("admin: 3DES(passwd,access-key)") 获取令牌。

Restful API请求到gateway后，根据请求URL的特性，gateway负责转发到相关的模块。

PG | ES | Redis 目前仅支持Data-Provider Service连接，暂不提供开放给其他系统服务直连获取数据。

### 3.认证方式(Check-in Check-out)

#### 3.1概述

在客户端访问UCSS API接口前，客户端必须向UCSS申请令牌。令牌申请需要在请求header中包含正确的用户名和密码，其中密码需要密文传送，使用3DES加密方式进行加密。

3DES加密模式: ECB ; 填充: pkcs5padding; 输出:base64

#### 3.2 Check-in

```
Request:
POST https://UCSS_IP:8443/qkact/v1/checkin HTTP/1.1
Content-Type: application/json
```

#### 3.3 Check-out

```
Request:
Delete https://UCSS_IP:8443/qkact/v1/checkin/{token} HTTP/1.1
```

### 4.可视化Restful API

#### 4.1概述

提供给可视化/第三方接口的数据。在Check-in成功后，可视化平台通过授权Token访问API获取页面所需要的数据。

#### 4.1.1基础接口

```
Request:
Post https://UCSS_IP:8443/qkact/v1/view/{datasource}/xxx HTTP/1.1
Content-Type: application/json
```

#### 4.1.2查询参数结构定义

- **4.1.2.1一维查询**: 查询ES或PG中的数据，一般获取的是列表数据或详情数据
- **4.1.2.2一维统计**: 以某一个特定的字段作为分组或聚合查询
- **4.1.2.3二维统计**: 以两个字段作为分组和聚合查询

### 4.2其他功能接口

### 5. Data-Provider Service API (sps-webService)

#### 5.1概述

API为其他服务如SWG、UCWI、DSAG等服务提供所需的数据，数据由sps从LES获取。

Data-Provider Service 使用代理转发模式。

- ES proxy: 通用的ES的代理
- 其他数据获取API: 特定数据的接口

#### 5.1.1 ES proxy

ES代理仅支持查询功能，修改、删除、归档、快照等功能不进行开放。目前仅支持GET和POST请求。

**5.1.1.1 Index查询接口**

```
Request:
Get https://UCSS_IP:8443/sps/v1/proxy/_cat/indices?v HTTP/1.1
Content-Type: application/json
```

**5.1.1.2 Search接口**

```
Request:
Post https://UCSS_IP:8443/sps/v1/proxy/{index}/_search HTTP/1.1
Get https://UCSS_IP:8443/sps/v1/proxy/{index}/_search?q=serialId:1 HTTP/1.1
Content-Type: application/json
```

Index类型说明：

| Index | 描述 |
|---|---|
| dlp-network,dlp-discovery,dlp-endpoint,dlp-mobile | DLP事件 |
| swg | SWG日志 |
| email-message | 邮件日志 |
| connect-log | 邮件连接日志 |
| itm-scores | ITM报告 |
| mobile-app-incident,mobile-mag-log | 移动事件 |
| ucwi | 行为日志 |
| redaction | 脱敏日志 |
| classify | 数据分类日志 |
| dlp-watermark | 终端打印暗水印记录 |
| dlp-dsag-discovery | 数据资产监控 |
