## 达梦数据库概述

### 达梦环境

- 达梦8：安装版，达梦8开发版，Windows Server 2008以上
- 达梦8：达梦容器化环境，ubuntu docker24 环境

### 使用说明

CLI中增加 `dbconf` 命令，仅在UCSS设备可用。

设置外置数据库：

```bash
$ dbconf set dameng 172.22.52.209 30236 SYSDBA <password>
```

参数说明：

```
$ dbconf set <type> <host> <port> <user> <password> [dbname]
```

| 参数 | 说明 |
|------|------|
| Type | dameng/mysql |
| Host | 数据库服务器IP |
| port | 数据库服务端口号 |
| User | 用户名 |
| Password | 数据库密码 |
| dbname | 数据库名称（仅Mysql时需要） |

### 端口说明

达梦默认端口是 **5236**。如果安装的是docker版本，启动之后宿主机是通过30236端口映射到5236端口的。实际端口取决于用户环境达梦数据库的部署模式。
